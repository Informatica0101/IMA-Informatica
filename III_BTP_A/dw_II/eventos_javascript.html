¿Qué es un Evento?

Un evento es una "señal" de que algo ha sucedido en la página web. JavaScript puede "escuchar" estas señales y reaccionar ejecutando un código específico.

Eventos de Ratón (Mouse)

  • onclick: Al hacer clic.
  • ondblclick: Doble clic.
  • onmouseover: Puntero encima.
  • onmouseout: Puntero sale.

Eventos de Teclado

Se activan cuando el usuario presiona o suelta una tecla.

  • onkeydown: Tecla presionada.
  • onkeyup: Tecla soltada.
  • onkeypress: Tecla mantenida (en desuso).

Eventos de Formulario

  • onfocus: El campo recibe el foco.
  • onblur: El campo pierde el foco.
  • onchange: El valor cambia.
  • onsubmit: Al enviar el formulario.

Eventos de Ventana

Relacionados con el navegador mismo.

  • onload: Cuando la página termina de cargar.
  • onresize: Al cambiar el tamaño de ventana.
  • onscroll: Al mover el scroll.

Manejadores de Eventos

Hay 3 formas de asignar un evento:

  1. 1. Atributo HTML: <button onclick="...">
  2. 2. Propiedad JS: elemento.onclick = ...
  3. 3. Escuchador (Recomendado): addEventListener()

addEventListener

Es la forma moderna y profesional. Permite asignar múltiples funciones al mismo evento sin sobrescribirlas.

btn.addEventListener("click", () => {
  console.log("Clic detectado");
});

El objeto 'event'

Cuando ocurre un evento, JS crea un objeto con información extra (ej: qué tecla se presionó, coordenadas del mouse).

window.onkeydown = (e) => {
  console.log("Presionaste: " + e.key);
}

Propagación (Bubbling)

Por defecto, un evento en un hijo se propaga hacia sus padres (como una burbuja). Se puede detener con event.stopPropagation().

Prevenir Acción: preventDefault

Cancela el comportamiento por defecto del navegador (ej: evita que un enlace te lleve a otra web o que un formulario se envíe).

Ejercicio: Cambio de Color

Escribe el código para que al presionar un botón, el fondo de la página cambie a color rojo.

btn.onclick = () => {
  document.body.style.background = "red";
}

Reto de Código

// Detectar movimiento del mouse

window.addEventListener("mousemove", (e) => {

  const x = e.clientX;

  const y = e.clientY;

  console.log(`Posición: ${x}, ${y}`);

});
