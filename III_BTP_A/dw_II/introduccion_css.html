¿Qué es CSS?

Cascading Style Sheets (Hojas de Estilo en Cascada).

Es un lenguaje que se usa para describir la apariencia y el formato de tu HTML.

Imagina que el HTML es el esqueleto y el CSS es la ropa y el maquillaje.

¿Por qué "en Cascada"?

Se llaman así porque las reglas pueden usarse unas sobre otras para sumarse y aplicarse a un mismo documento.

Si defines que todos los párrafos son azules, pero luego dices que uno en específico es rojo, CSS aplicará el cambio basado en prioridades.

Forma vs. Función

Existen dos razones principales para separar el CSS del HTML:

  • Reutilización: Aplicar el mismo formato a diversos elementos sin repetir código.
  • Mantenimiento: Cambiar la apariencia de todo un sitio web desde un solo archivo centralizado.

Estilos en Línea (Inline)

Se aplican directamente en la etiqueta HTML usando el atributo style.

<p style="color:red">¡Soy rojo!</p>

¡No recomendado!

Obliga a repetir código y dificulta cambios globales.

Estilos Internos

Se colocan dentro de las etiquetas <style> en el encabezado del HTML.

<head>
  <style>
    p { color: purple; }
  </style>
</head>

Estilos Externos

Es la mejor forma. Se crea un archivo .css independiente y se enlaza al HTML.

Permite que múltiples páginas HTML compartan el mismo diseño, ahorrando tiempo y código.

¿Cómo se enlaza?

Se usa la etiqueta <link> dentro del <head>.

  • type: Siempre "text/css".
  • rel: Siempre "stylesheet".
  • href: La ruta del archivo (ej: "style.css").

Ejemplo de etiqueta link

<link type="text/css" rel="stylesheet"
      href="stylesheet.css" />

Notarás que termina en /> porque es una etiqueta autocerrable.

Etiquetas Autocerrables

Algunas etiquetas no tienen una etiqueta de cierre separada (como </p>).

  • <link />: Para enlaces.
  • <img />: Para imágenes.
  • <br />: Para saltos de línea.

Sintaxis Básica

selector {
  propiedad: valor;
}
  • Selector: A qué elemento aplicamos el estilo.
  • Propiedad: Qué característica queremos cambiar.
  • Valor: El nuevo estado de esa propiedad.

Unidades de Medida

Para definir tamaños, CSS utiliza diferentes unidades:

px: Píxeles (Fijo).
%: Porcentaje (Relativo).
em: Basado en fuente (Relativo).

Background Color

Define el color de fondo de un elemento.

div {
  background-color: yellow;
}

Height y Width

Controlan las dimensiones de los contenedores o elementos.

img {
  height: 100px;
  width: 200px;
}
