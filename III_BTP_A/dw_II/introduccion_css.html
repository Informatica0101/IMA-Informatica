¿Qué es CSS?

Cascading Style Sheets (Hojas de Estilo en Cascada).

Es un lenguaje que se usa para describir la apariencia y el formato de tu HTML.

¿Por qué "en Cascada"?

Se llaman así porque las reglas pueden usarse unas sobre otras para sumarse y aplicarse a un mismo documento.

Si indicas que todos los párrafos deben tener la fuente azul, pero luego señalas que un párrafo específico debe ser rojo, ¡CSS lo puede hacer!

¿Por qué separar la forma de la función?

Existen dos razones principales para separar el CSS (Formato) del HTML (Estructura):

  • Reutilización: Puedes aplicarle el mismo formato a diversos elementos sin tener que volver a escribir el código.
  • Apariencia global: Puedes aplicarle un formato parecido a varias páginas desde un solo archivo CSS.

Estilos en Línea (Inline)

Se aplican directamente en la etiqueta HTML usando el atributo style.

<p style="color:red">Red font!</p>

¡No es la mejor forma!

Si quieres cambiar el estilo de varios elementos, tendrías que cambiar cada etiqueta una por una.

Estilos Internos

Se colocan justo en el mismo archivo de código HTML, dentro de las etiquetas <style>.

<head>
  <style>
    p { color: purple; }
  </style>
</head>

Estilos Externos (Fuera de línea)

Es la forma más eficiente. Se escribe el código en un archivo aparte (con extensión .css) y se conecta al HTML.

Permite que un único archivo controle el diseño de todo tu sitio web.

¿Cómo se enlaza?

Se agrega la etiqueta <link> entre las etiquetas <head> de tu página HTML.

  • type: Siempre debe ser "text/css".
  • rel: Siempre debe ser "stylesheet".
  • href: La dirección de tu archivo CSS.

Sintaxis de enlace

<link type="text/css" rel="stylesheet"
      href="stylesheet.css" />

Notarás que termina en /> porque es una etiqueta autocerrable.
