¿Qué es CSS?

Cascading Style Sheets: Es el lenguaje que define cómo se deben mostrar los elementos HTML. Separa el contenido (HTML) de la presentación (Diseño).

La Cascada

El término "Cascada" significa que si un elemento tiene varias reglas aplicadas, el navegador seguirá un orden de prioridad para decidir cuál usar (normalmente la última regla escrita).

Regla CSS: Estructura

Selector { propiedad: valor; }

h1 {
  color: blue;
  font-size: 20px;
}

Formas de Aplicar CSS

  • Inline: Dentro del atributo style de la etiqueta.
  • Interno: Dentro de etiquetas <style> en el head.
  • Externo: En un archivo .css aparte (El mejor método).

Selectores Básicos

Etiqueta (h1)
Clase (.mi-clase)
ID (#mi-id)

Ejercicio: Selector de Clase

Si tienes un párrafo con class="destacado", ¿cómo escribirías la regla CSS para ponerle el fondo amarillo?

.destacado { background: yellow; }

Colores en CSS

Podemos usar:

  • Nombres (red)
  • Hex (#FF0000)
  • RGB (255,0,0)

Comentarios en CSS

Sirven para documentar el código. El navegador los ignora.

/* Este es un comentario */

Unidades de Medida

  • px: Píxeles (Tamaño fijo).
  • %: Porcentaje (Relativo al padre).
  • rem / em: Relativo al tamaño de fuente.

Importancia de la Herencia

Muchas propiedades (como el color o la fuente) se heredan de los padres a los hijos automáticamente.

El Modelo de Caja

Es el concepto más importante: cada elemento es una caja con contenido, padding, borde y margen.

Reto de Código

<link rel="stylesheet" href="estilos.css">

/* En el archivo CSS */

body {

  background-color: #f0f0f0;

  font-family: Arial, sans-serif;

}
