¿Qué es CSS?

Cascading Style Sheets (Hojas de Estilo en Cascada).

Es un lenguaje que se usa para describir la apariencia y el formato de tu HTML.

Separa la estructura (HTML) de la presentación (CSS).

¿Por qué "en Cascada"?

Se llaman así porque las reglas pueden usarse unas sobre otras para sumarse y aplicarse a un mismo documento.

Ejemplo: Si indicas que todos los párrafos deben ser azules, pero señalas que un párrafo específico debe ser rojo, ¡CSS lo puede hacer siguiendo un orden de prioridad!

Sintaxis Básica

p {
  color: red;
  font-size: 16px;
}
  • Selector: p (a qué elemento aplicamos estilo).
  • Propiedad: color (qué queremos cambiar).
  • Valor: red (el nuevo estado).

Forma vs. Función

¿Por qué separar el contenido (HTML) del formato (CSS)?

  • Reutilización: Aplicar el mismo formato a diversos elementos sin repetir código.
  • Mantenimiento: Cambiar la apariencia de todo un sitio web desde un solo archivo.
  • Eficiencia: Reduce el tamaño de los archivos HTML.

Mejor fuera de línea

Existen varias formas de aplicar CSS, pero la mejor es mediante un archivo externo.

Inline (No recomendado)

<p style="color:red">...</p>

Obliga a repetir código en cada etiqueta.

Externo (Recomendado)

<link rel="stylesheet" href="style.css">

Un solo lugar para todos los cambios.

¡Pon a prueba tus conocimientos!

Ejercicio Práctico

Objetivo: Enlazar CSS a HTML

  1. Crea un archivo llamado index.html.
  2. Crea un archivo llamado stylesheet.css.
  3. En el HTML, usa la etiqueta <link> para conectar el CSS.
  4. Define que todos los <span> tengan color rojo y fuente cursiva en el CSS.
  5. Visualiza los cambios en tu navegador.
