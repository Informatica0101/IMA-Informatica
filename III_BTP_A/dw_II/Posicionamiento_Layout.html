El Modelo de Caja (Box Model)

En CSS, cada elemento HTML se considera una caja rectangular. Entender cómo se calculan sus dimensiones es clave para el diseño.

Los cuatro componentes principales son: Content, Padding, Border y Margin.

Content y Padding

Content (Contenido):

Donde aparece el texto o las imágenes del elemento.

Padding (Relleno):

Espacio transparente entre el contenido y el borde.

Border y Margin

Border (Borde):

Línea que rodea el relleno y el contenido.

Margin (Margen):

Espacio transparente fuera del borde, separa al elemento de sus vecinos.

Jerarquía del Modelo de Caja

Margin
Border Padding
CONTENT

El tamaño total = Content + Padding + Border + Margin

La propiedad Position

Define cómo se ubica un elemento en el flujo del documento.

  • Static: Posición por defecto, sigue el flujo normal.
  • Relative: Se desplaza respecto a su posición original.
  • Absolute: Se posiciona respecto a su ancestro más cercano.
  • Fixed: Se fija respecto a la ventana del navegador.

Layout con Floats

La propiedad float permite empujar elementos a la izquierda o derecha de su contenedor.

Se utilizaba históricamente para crear columnas. Los valores comunes son left, right y none.

Nota: Al usar floats, es necesario usar "clear" para evitar que el contenedor colapse.

Display: Block vs Inline

Display: Block

Ocupa todo el ancho disponible. Empieza en una nueva línea (ej: div, p, h1).

Display: Inline

Ocupa solo el espacio necesario. No rompe la línea (ej: span, a, img).

Z-Index: El Orden de las Capas

Controla qué elementos aparecen "encima" de otros cuando hay superposición.

Solo funciona en elementos con una posición distinta a static. Los valores más altos se dibujan al frente.

Práctica en VSCode: Modelo de Caja

Crea un archivo index.html y añade este estilo básico:

<style>

.caja {

width: 200px;

padding: 20px;

border: 5px solid blue;

margin: 10px;

}

</style>

<div class="caja">Hola Mundo</div>
