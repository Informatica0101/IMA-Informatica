7. Ramificación en CSS

Hasta ahora hemos aplicado estilos directamente a etiquetas generales (como p o div). Sin embargo, a veces necesitamos mayor precisión.

CSS nos permite "ramificar" y seleccionar elementos específicos utilizando atributos HTML especiales denominados Clases e Identificadores (IDs).

7.1 Clases en CSS

Las clases son sumamente útiles cuando tenemos un estilo específico que queremos aplicar a múltiples elementos HTML a lo largo de una misma página.

En el HTML: Se asignan usando el atributo class="nombre-clase".

En el CSS: Se identifican anteponiendo siempre un punto (.) antes del nombre.

Ejemplo Práctico: Uso de Clases

Código HTML:
<p class="destacado">Texto Uno</p>
<span>Texto Común y Corriente</span>
<h4 class="destacado">Título Dos</h4>
Código CSS:
.destacado {
  color: #db2777;
  font-weight: bold;
}
Resultado en el Navegador:
mi-pagina.html

Texto Uno

Texto Común y Corriente

Título Dos

7.2 Identificadores (IDs)

A diferencia de las clases, un identificador es una etiqueta exclusiva orientada a un único elemento dentro del documento HTML.

Restricción: No se permite repetir el mismo id en más de un elemento de la página.

En el CSS: Se seleccionan anteponiendo el símbolo de numeral (#).

Ejemplo Práctico: Uso de IDs

Código HTML:
<div id="menu-principal">
  Inicio | Productos | Contacto
</div>
Código CSS:
#menu-principal {
  background-color: #1f2937;
  color: #ffffff;
  padding: 10px;
}
Resultado en el Navegador:
sitio-web.com
Inicio  |  Productos  |  Contacto

[Resto del contenido de la página]

7.4 Selectores de Pseudoclase

Una pseudoclase es una propiedad que se añade a los selectores para definir un estado especial del elemento HTML.

Permite aplicar formatos basados en acciones del usuario o en la estructura del documento sin necesidad de usar JavaScript.

Sintaxis general: selector:pseudoclase { propiedad: valor; }

7.5 Controlando Enlaces (a)

Podemos cambiar la apariencia de las etiquetas de hipervínculo (<a>) según su estado dinámico:

• a:link : El aspecto por defecto del enlace cuando no ha sido visitado.

• a:visited : Cambia el estilo una vez que el usuario ya hizo clic en él.

• a:hover : ¡Muy importante! Se activa justo cuando el cursor del mouse se posiciona encima del enlace.

Ejemplo Práctico: El estado de Hover

Código HTML:
<a class="boton" href="#">Enviar</a>
Código CSS:
.boton { color: black; border: 1px solid black; }
.boton:hover {
  background-color: #ec4899;
  color: white;
}
Simulación de Navegador (Pasa el cursor por encima):
enlaces.html
Enviar
