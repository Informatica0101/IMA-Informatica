El Concepto Base

En el diseño web, TODO es una caja rectangular. El navegador dibuja cada elemento (h1, p, img, div) como una caja y las apila una tras otra.

Partes de la Caja

  • Content: El contenido real (texto/imagen).
  • Padding: Espacio interno (entre contenido y borde).
  • Border: La línea que rodea al padding.
  • Margin: Espacio externo (separa la caja de otras).

Demostración Visual

Dimensiones: width y height

Por defecto, estas propiedades definen SOLO el tamaño del área de contenido. El padding y el borde se suman al tamaño final.

width: 300px; height: 150px;

Cálculo del Ancho Total

Si una caja tiene width: 100px, padding: 10px y border: 5px...

Ancho Total = 5+10+100+10+5 = 130px

box-sizing: border-box

Esta propiedad cambia la forma en que se calcula el tamaño. Si pones 100px, la caja medirá 100px incluyendo el padding y el borde.

Es la mejor práctica y se usa casi siempre.

Márgenes Automáticos

Usamos margin: auto junto con un ancho definido para centrar un bloque horizontalmente.

div {
  width: 50%;
  margin: 0 auto;
}

Dimensiones Máximas y Mínimas

  • min-width / min-height: Evita que la caja sea demasiado pequeña.
  • max-width / max-height: Evita que la caja desborde en pantallas grandes.

Desbordamiento: overflow

Define qué pasa si el contenido es más grande que la caja.

  • visible
  • hidden (Corta)
  • scroll (Barras)

Padding vs Margin

Padding: Afecta al fondo del elemento.
Margin: No tiene fondo (es transparente).

Sintaxis Corta (Shorthand)

Puedes definir los 4 lados en una sola línea (Arriba, Derecha, Abajo, Izquierda).

padding: 10px 20px 5px 30px;

Reto de Código

* { box-sizing: border-box; }

.contenedor {

  max-width: 1200px;

  margin: 0 auto;

  padding: 0 15px;

}
