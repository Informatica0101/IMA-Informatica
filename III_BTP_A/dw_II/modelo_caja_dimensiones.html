El Modelo de Caja (Box Model)

En CSS, cada elemento HTML se considera como una caja rectangular.

Este modelo permite controlar el espacio, el borde y el contenido de cada elemento de forma independiente.

Componentes de la Caja

  • Contenido: Donde aparece el texto o imágenes.
  • Padding (Relleno): Espacio entre el contenido y el borde.
  • Borde: Línea que rodea el padding y el contenido.
  • Margen: Espacio exterior que separa la caja de otras cajas.

Esquema Visual

CONTENIDO

Márgenes (Margin)

Controla el espacio fuera del elemento.

margin-top: 20px;
margin-right: 15px;
margin-bottom: 20px;
margin-left: 15px;

Relleno (Padding)

Espacio entre el contenido y el borde. Afecta el tamaño visual del fondo del elemento.

padding: 10px 20px 10px 20px;
background-color: yellow;

Dimensiones del Contenido

Propiedades width (ancho) y height (alto).

Por defecto, en el modelo estándar, el ancho total es:
Ancho = width + padding + border

Cambiando el Cálculo: Border-Box

Es la mejor práctica moderna. Incluye el padding y el borde dentro del ancho definido.

* {
  box-sizing: border-box;
}

Unidades Comunes

  • px: Píxeles (Fijo).
  • %: Porcentaje relativo al contenedor padre.
  • vh / vw: Relativo al tamaño de la ventana (Viewport).
  • auto: El navegador calcula el espacio automáticamente.
