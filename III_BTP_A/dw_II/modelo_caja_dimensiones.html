El Modelo de Caja (Box Model)

En CSS, cada elemento HTML se considera como una caja rectangular.

Este modelo permite controlar el espacio, el borde y el contenido de cada elemento de forma independiente.

Componentes de la Caja

  • Contenido: Donde aparece el texto o imágenes.
  • Padding (Relleno): Espacio entre el contenido y el borde.
  • Borde: Línea que rodea el padding y el contenido.
  • Margen: Espacio exterior que separa la caja de otras cajas.

Márgenes (Margin)

Controla el espacio fuera del elemento.

margin-top: 20px;
margin-right: 15px;
margin-bottom: 20px;
margin-left: 15px;

Abreviado: margin: 20px 15px; (Arriba/Abajo Izq/Der)

Relleno (Padding)

Espacio entre el contenido y el borde. Afecta el tamaño visual del fondo del elemento.

padding: 10px 20px 10px 20px;
background-color: yellow;

El color de fondo se extiende por el área del padding.

Dimensiones del Contenido

Propiedades width (ancho) y height (alto).

Por defecto, en el modelo estándar, el ancho total es:
Ancho = width + padding + border

Cambiando el Cálculo: Border-Box

Es la mejor práctica moderna. Incluye el padding y el borde dentro del ancho definido.

* {
  box-sizing: border-box;
}

Esto evita que los elementos se "rompan" al añadir padding.

Unidades Comunes

  • px: Píxeles (Fijo).
  • %: Porcentaje relativo al contenedor padre.
  • vh / vw: Relativo al tamaño de la ventana (Viewport).
  • auto: El navegador calcula el espacio automáticamente.

Truco: Centrado Horizontal

Para centrar un bloque, dale un ancho fijo y márgenes automáticos a los lados.

width: 50%;
margin: 0 auto;

Límites: Max y Min

Permiten que el diseño sea flexible pero controlado.

max-width: 1200px;

No crecerá más allá de 1200px aunque la pantalla sea enorme.

min-height: 300px;

Garantiza un espacio mínimo incluso si no hay mucho texto.

Desbordamiento (Overflow)

Controla qué pasa cuando el contenido es más grande que la caja.

  • visible: El contenido sale de la caja (default).
  • hidden: Se recorta lo que sobra.
  • scroll: Añade barras de desplazamiento siempre.
  • auto: Añade barras solo si es necesario.

Ejercicio: La Tarjeta Perfecta

Crea un archivo HTML con una caja que tenga:

  • Ancho de 300px y Alto de 200px.
  • Fondo celeste y borde azul de 2px.
  • Padding de 20px para que el texto no toque el borde.
  • Margen de 50px arriba para separarlo del título.
