Uso de Tablas en Web

Las tablas (<table>) se utilizan exclusivamente para mostrar datos tabulares (filas y columnas). Antiguamente se usaban para diseño, pero hoy en día eso es una mala práctica.

Etiquetas Básicas

  • <tr>: Fila (Table Row).
  • <th>: Encabezado (Table Header).
  • <td>: Celda (Table Data).
  • <caption>: Título de tabla.

Combinar Celdas

  • colspan: Une varias columnas horizontalmente.
  • rowspan: Une varias filas verticalmente.
<td colspan="2">Celda doble</td>

Estilo: border-collapse

Por defecto, las celdas tienen un espacio entre ellas. El valor collapse las une en una sola línea delgada.

table { border-collapse: collapse; }

Secciones de Tabla

Para tablas largas, usamos etiquetas semánticas:

  • <thead>: Parte superior (cabecera).
  • <tbody>: Cuerpo de datos.
  • <tfoot>: Parte inferior (totales).

Ejercicio: Tabla de Notas

Escribe el código para una tabla que tenga:

  • Una fila de encabezado (Materia, Nota).
  • Tres filas de datos.
  • Bordes colapsados de 1px negro.

Columnas de Texto (Multicolumn)

CSS permite dividir un bloque de texto en columnas tipo periódico sin usar tablas.

.periodico {
  column-count: 3;
  column-gap: 20px;
}

column-rule

Dibuja una línea divisoria entre las columnas de texto, similar a un borde.

column-rule: 1px solid gray;

Efecto Cebra (Zebra Stripes)

Mejora la lectura alternando colores en las filas. Usamos el selector pseudo-clase :nth-child.

tr:nth-child(even) { background: #f2f2f2; }

Tablas Responsivas

Las tablas suelen romperse en móviles. La solución más simple es envolver la tabla en un div con overflow-x: auto.

Alineación en Celdas

Usamos text-align para horizontal y vertical-align para vertical (top, middle, bottom).

Reto de Código

<table>

  <tr>

    <th>ID</th> <th>User</th>

  </tr>

  <tr>

    <td>1</td> <td>Admin</td>

  </tr>

</table>
