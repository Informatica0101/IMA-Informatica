¿Qué es JavaScript?

Es un lenguaje de programación que permite añadir interactividad y dinamismo a las páginas web. Mientras el HTML es la estructura y el CSS es el diseño, JS es el comportamiento.

Variables: let y const

Contenedores para guardar datos.

  • const: Para valores que NO cambian.
  • let: Para valores que pueden cambiar.
const nombre = "Isemed";
let puntos = 10;

Tipos de Datos

  • String: Texto ("Hola")
  • Number: Números (10, 3.14)
  • Boolean: Lógica (true, false)
  • Array: Listas [1, 2, 3]

Funciones

Bloques de código reutilizables que realizan una tarea específica.

function saludar() {
  console.log("¡Hola!");
}

Operadores

  • Aritméticos: +, -, *, /
  • Comparación: ==, ===, >, <, !=
  • Lógicos: && (Y), || (O), ! (No)

Ejercicio: Cálculo Simple

Escribe el código para una variable precio de 100 y una variable impuesto de 0.15. Calcula el total y muéstralo con console.log.

Condicional IF

Permite tomar decisiones en el código.

if (nota >= 70) {
  console.log("Aprobado");
} else {
  console.log("Reprobado");
}

El DOM y JS

JavaScript usa el DOM para cambiar el HTML. El método más común es document.getElementById().

Alertas y Mensajes

Formas rápidas de interactuar con el usuario:

  • alert()
  • prompt()
  • confirm()

Eventos Básicos

JS reacciona a lo que hace el usuario:

onclick
onmouseover
onkeydown

La Consola

Presiona F12 en tu navegador. Es la herramienta principal para depurar errores y probar código rápidamente.

Reto de Código

const btn = document.querySelector("#miboton");

btn.onclick = function() {

  alert("¡Hiciste clic!");

  btn.style.backgroundColor = "red";

};
