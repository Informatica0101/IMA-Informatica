¿Qué es JavaScript?

JavaScript (JS) es un lenguaje de programación ligero e interpretado, fundamental para el desarrollo web moderno.

  • Permite crear contenido dinámico e interactivo en sitios web.
  • Se ejecuta principalmente en el lado del cliente (navegador del usuario).
  • Es compatible con HTML y CSS para construir experiencias web completas.
  • También se usa en el lado del servidor (Node.js), aplicaciones móviles y de escritorio.

Variables en JavaScript

Las variables son contenedores para almacenar datos. JS tiene tres palabras clave para declararlas:

  • var: Declaración antigua. Alcance de función o global. Puede ser redeclarada y reasignada.
  • let: Introducido en ES6. Alcance de bloque. Puede ser reasignada, pero no redeclarada en el mismo alcance.
  • const: Introducido en ES6. Alcance de bloque. No puede ser reasignada ni redeclarada. Debe inicializarse.

Ejemplo: Declaración de Variables

Sintaxis del Código:

var nombre = "Ana"; let edad = 30; const PI = 3.14159; console.log("Nombre: " + nombre); console.log("Edad inicial: " + edad); edad = 31; // Válido con let, reasigna el valor console.log("Nueva edad: " + edad); // PI = 3.14; // Error: No se puede reasignar const

Tipos de Datos Fundamentales

JavaScript maneja varios tipos de datos para representar información:

  • String: Texto (ej. "Hola mundo").
  • Number: Números enteros o decimales (ej. 10, 3.14).
  • Boolean: Valores lógicos true o false.
  • Array: Colección ordenada de elementos (ej. [1, 2, 3]).
  • Object: Colección de pares clave-valor (ej. {nombre: "Juan", edad: 25}).

Ejemplo: Tipos de Datos

Sintaxis del Código:

let mensaje = "Hola"; // String let cantidad = 100; // Number let activo = true; // Boolean let colores = ["rojo", "verde"]; // Array let usuario = {id: 1, nombre: "Maria"}; // Object console.log("Tipo de mensaje: " + typeof mensaje); console.log("Tipo de cantidad: " + typeof cantidad); console.log("Tipo de activo: " + typeof activo); console.log("Tipo de colores: " + typeof colores); // Arrays son de tipo 'object' console.log("Tipo de usuario: " + typeof usuario);

Estructuras de Control: Condicionales

Permiten ejecutar código basado en ciertas condiciones.

  • if / else if / else: Para ejecutar bloques de código condicionalmente.
  • switch: Para evaluar una expresión contra múltiples casos.

Ejemplo: Condicional if/else

Sintaxis del Código:

let hora = 14; // Prueba con diferentes horas, ej. 9, 14, 20 if (hora < 12) { console.log("Buenos días"); } else if (hora < 18) { console.log("Buenas tardes"); } else { console.log("Buenas noches"); }

Estructuras de Control: Bucles

Permiten repetir un bloque de código múltiples veces.

  • for: Para bucles con un número conocido de iteraciones.
  • while: Para bucles que continúan mientras una condición sea verdadera.
