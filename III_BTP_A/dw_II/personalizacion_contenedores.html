Color de fondo, Dimensionamiento y Estilos de Enlaces

Lic. Pablo Antonio Peña Mancia

Fecha: 17 y 24 de mayo de 2026

Grado: III BTP Informática Sección "A"

Color de Fondo

La propiedad background-color establece el color de fondo de un elemento.

Sintaxis:

elemento { background-color: valor; }

Valores: nombres (red), hexadecimal (#FF0000), RGB (rgb(255,0,0)).

Altura y Ancho

Determinan el espacio que ocupa el contenido de la caja.

  • width: Anchura del elemento.
  • height: Altura del elemento.

Ejemplo: div { width: 50%; height: 200px; }

El Modelo de Caja

Todo elemento es una caja compuesta por capas:

  • Content: El texto o imagen.
  • Padding: Espacio entre contenido y borde.
  • Border: Línea que rodea el padding.
  • Margin: Espacio fuera del borde.

Bordes en CSS

La propiedad border es una abreviación (shorthand) para tres valores:

border: [grosor] [estilo] [color];

Ejemplo: border: 2px solid blue;

Estilos: solid, dashed, dotted, double.

Bordes Redondeados

Permite suavizar las esquinas de los contenedores.

border-radius: 15px;

Si se usa el 50%, el elemento se vuelve un círculo (si es cuadrado).

Ejercicio 1: La Caja

Escribe el código CSS para un section con:

  • Fondo gris claro.
  • Ancho de 400px y altura de 100px.
  • Borde rojo sólido de 5px.
  • Esquinas redondeadas de 10px.

Personalización de Enlaces

Por defecto, los enlaces son azules y subrayados. Podemos cambiar esto con CSS.

a { color: green; text-decoration: none; }

text-decoration: none; elimina el subrayado.

Estados del Enlace

:link - Estado normal.

:visited - Enlace ya visitado.

:hover - Al pasar el mouse.

:active - Al hacer clic.

Decoración de Texto

Controla las líneas decorativas en el texto.

  • underline: Subrayado.
  • overline: Línea superior.
  • line-through: Tachado.
  • none: Sin decoración.

Ejemplo: Botón de Navegación

a.boton {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  text-decoration: none;
  border-radius: 5px;
}

Impacto en la Usabilidad

Un enlace debe parecer un enlace. Quitar el subrayado requiere compensar con color o negrita para que el usuario sepa que es clicable.

Consejo: Usa :hover para dar feedback visual al usuario.

Ejercicio 2: Menú

Diseña en tu cuaderno un menú con 3 enlaces. Escribe el CSS para que al pasar el mouse (:hover), el fondo cambie a amarillo y el texto a negro.
