Color de fondo, Dimensionamiento y Estilos de Enlaces

Clase: Diseño Web I

Fecha: 17 y 24 de mayo de 2026

Tema: Personalización Avanzada

Color de Fondo

La propiedad background-color permite definir el ambiente visual de un contenedor.

Podemos usar nombres básicos, códigos Hexadecimales (#ffffff) o formatos funcionales como RGB y HSL para mayor precisión.

Dimensionamiento (Box Model)

Todo elemento en HTML es una caja. Sus dimensiones principales se controlan con width y height.

  • Width (Ancho): Controla el espacio horizontal.
  • Height (Alto): Controla el espacio vertical.

Unidades de Medida

Es vital elegir la unidad correcta según el dispositivo:

Absolutas: px (píxeles) para tamaños fijos.

Relativas: % (porcentaje) para diseños que se adaptan a la pantalla.

Bordes (Properties)

La propiedad border es una abreviación que define grosor, estilo y color al mismo tiempo.

border: 2px solid #0d6efd;

Bordes Redondeados

Con border-radius podemos suavizar las esquinas o crear círculos perfectos.

Un border-radius: 50% aplicado a un elemento cuadrado lo transformará automáticamente en un círculo.

Ejercicio 1: Contenedores

Dibuja en tu cuaderno una caja y escribe el código CSS para que tenga fondo verde, un borde doble negro y esquinas de 20px.

Pista: border-style: double;

Estados de los Enlaces

Los enlaces (<a>) tienen estados interactivos conocidos como pseudo-clases.

  • :link - Estado inicial.
  • :visited - Enlace ya clicado.
  • :hover - Mouse encima.
  • :active - Momento del clic.

Decoración de Texto

La propiedad text-decoration nos permite quitar el subrayado clásico de los enlaces.

Sintaxis: text-decoration: none;

Se usa mucho para menús de navegación modernos.

Cambiando Colores de Enlace

Es una buena práctica cambiar el color de un enlace cuando el usuario pasa el mouse sobre él (feedback).

a:hover {
  color: orange;
  font-weight: bold;
}

Sombras (Box-shadow)

Agrega profundidad a tus contenedores con sombras personalizadas.

Definimos desplazamiento horizontal, vertical, desenfoque (blur) y color.

Ejemplo: box-shadow: 0 4px 10px rgba(0,0,0,0.1);

Layout y Alineación

El dimensionamiento correcto permite que los contenedores se alineen correctamente en la cuadrícula del sitio.

Recuerda: El margen (margin) agrega espacio fuera de la caja.
