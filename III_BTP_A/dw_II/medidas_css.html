Unidades de Medida

En CSS, necesitamos especificar el tamaño de diversos elementos: fuentes, márgenes, anchos y altos.

Existen dos tipos principales de unidades: Absolutas (fijas) y Relativas (flexibles).

Píxeles (px)

Un píxel es un punto físico en la pantalla de la computadora.

Uso: Son geniales cuando quieres que el usuario vea exactamente lo mismo que tú diseñaste. Sin embargo, no se adaptan bien a pantallas muy diferentes (como smartphones).

Sintaxis con px

p {
  font-size: 16px;
  width: 300px;
}

Define un tamaño fijo que no cambiará independientemente del dispositivo.

Unidades Relativas: em

Un em es igual al tamaño de fuente predeterminado de la pantalla o del elemento padre.

Ventaja: Son ideales para smartphones porque se escalan proporcionalmente. "1em es el tamaño normal, 2em es el doble, 0.5em es la mitad".

px vs em

Píxeles (px):

  • Medida absoluta y fija.
  • Precisión exacta.
  • Dificultad en móviles.

Em (em):

  • Medida relativa y flexible.
  • Escalabilidad proporcional.
  • Ideal para diseño adaptativo.

Unidad Relativa: Porcentaje (%)

Se utiliza comúnmente para anchos (width) y alturas (height), tomando como referencia el elemento contenedor.

div {
  width: 50%;
}

El contenedor ocupará la mitad del espacio disponible de su padre.

Fuentes en CSS

CSS incluye familias de fuentes genéricas para asegurar que el usuario vea algo similar a tu intención:

  • serif: Con remates decorativos (ej: Times New Roman).
  • sans-serif: Sin adornos, limpia (ej: Arial).
  • cursive: Estilo hecho a mano alzada.

Definiendo Tipografías

h1 {
  font-family: Verdana, sans-serif;
}

Si el sistema no tiene "Verdana", usará cualquier fuente "sans-serif" disponible.

Combinando Estilos

Un solo selector puede tener muchas propiedades agrupadas:

p {
  font-family: Arial;
  color: blue;
  font-size: 24px;
}

¡No olvides los puntos y comas (;)!
