Importancia de las Unidades

En el diseño web moderno, las pantallas tienen miles de tamaños diferentes. Usar la unidad de medida correcta es la diferencia entre una web profesional y una que se rompe al abrirla en un móvil.

Medidas Absolutas

Tienen un tamaño fijo y no cambian. El ejemplo principal es el Píxel (px).
Ideales para bordes o elementos que no deben estirarse.

border: 2px solid black;

Medidas Relativas

Se adaptan según el tamaño de otro elemento (el padre, la fuente o la pantalla).

  • %: Porcentaje.
  • em / rem: Relativos al texto.
  • vh / vw: Relativos a la ventana.

Porcentaje (%)

Define el tamaño relativo al contenedor padre. Si el padre mide 1000px y el hijo 50%, el hijo medirá 500px.

50%

La unidad rem

Es la unidad más recomendada para fuentes. Se basa en el tamaño de la raíz (html). Por defecto, 1rem = 16px.

font-size: 1.5rem; /* Sería 24px */

Ejercicio: Cálculo rem

Si cambias el tamaño base del HTML a 20px, ¿cuánto medirá un título con 3rem?

Respuesta: 60 píxeles.

Viewport Units (vh / vw)

Se basan en el tamaño de la ventana del navegador (lo que el usuario ve).

  • 100vw: Todo el ancho de la pantalla.
  • 100vh: Todo el alto de la pantalla.

Diferencia em vs rem

em: Relativo al padre inmediato (puede causar efectos en cadena).
rem: Relativo al HTML (siempre es el mismo valor en toda la web).

Uso de fr (Fraction)

Unidad exclusiva de CSS Grid. Representa una fracción del espacio libre en el grid.

grid-template-columns: 1fr 2fr;

Medidas de Tiempo

Usadas en animaciones y transiciones.

  • s (Segundos)
  • ms (Milisegundos)

1000ms = 1s

Medidas de Ángulo

Usadas para rotar elementos (transform: rotate).

transform: rotate(45deg);

Reto de Código

section {

  min-height: 100vh; /* Ocupa toda la pantalla */

  width: 100%;

  font-size: 1.2rem;

  padding: 2em; /* Relativo a su propia fuente */

}

Asignación

DISEÑO RESPONSIVO

Crear un encabezado (hero section) que ocupe exactamente el 100% de la altura de la pantalla (100vh). El título principal debe tener un tamaño de 4rem y el subtítulo 1.5rem. Probar cómo se ve en el navegador al cambiar el tamaño de la ventana.
