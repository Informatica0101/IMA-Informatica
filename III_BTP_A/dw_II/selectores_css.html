¿Qué es un Selector CSS?

Los selectores son la parte de la regla CSS que determina a qué elementos HTML se les aplicará el estilo definido.

Permiten apuntar con precisión a los elementos que queremos modificar dentro del DOM (Document Object Model).

Selectores Simples (De Etiqueta)

Selecciona todos los elementos que tienen el nombre de una etiqueta HTML específica.

Uso en HTML

<p>Contenido del párrafo</p>

Uso en CSS

p {
  color: red;
}

Este ejemplo aplica el color rojo a todos los párrafos del documento.

Selectores Simples (De Clase)

Selecciona elementos con un atributo class específico. Se define con un punto (.).

Uso en HTML

<div class="texto-destacado">

Uso en CSS

.texto-destacado {
  font-weight: bold;
}

Las clases pueden usarse en múltiples elementos diferentes.

Selectores Simples (De ID)

Selecciona un único elemento con un atributo id único. Se define con (#).

Uso en HTML

<header id="cabecera">

Uso en CSS

#cabecera {
  background: blue;
}

A diferencia de las clases, un ID debe ser único en toda la página.

Comparativa: Etiqueta vs Clase vs ID

Etiqueta

Afecta a todos los elementos del mismo tipo.

Clase

Reutilizable en varios elementos diferentes.

ID

Único. Identifica un solo elemento específico.

Selectores Múltiples (Agrupados)

Si quieres aplicar el mismo estilo a varios elementos diferentes, puedes agrupar sus selectores separándolos con una coma (,).

Esto ayuda a mantener el código CSS más corto, eficiente y fácil de leer.

Ejemplo de Agrupación

Uso en HTML

<h1>Título</h1>
<p>Párrafo</p>

Uso en CSS

h1, p {
  color: #2c3e50;
}

Resultado: Todos los elementos en la lista compartirán el mismo estilo.

Selector Global (Universal)

El selector universal se representa con un asterisco (*).

Selecciona absolutamente todos los elementos del documento HTML al mismo tiempo.

Uso del Selector Global

Se utiliza para aplicar una regla a todos los elementos del documento sin excepción.

Uso en HTML

Afecta a todo el DOM

Uso en CSS

* {
  margin: 0;
  padding: 0;
}

Ventajas de Optimizar Selectores

  • Reducción de código: Evitas repetir las mismas reglas (DRY).
  • Mantenimiento sencillo: Al cambiar un valor, se actualizan todos los elementos agrupados.
  • Velocidad: Archivos CSS más ligeros cargan más rápido.
