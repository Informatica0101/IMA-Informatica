¿Qué es un Selector?

Es la parte de una regla CSS que indica al navegador qué elemento o elementos HTML deben recibir los estilos definidos.

Selectores Básicos

  • Universal (*): Selecciona TODO.
  • Etiqueta (p, h1): Por nombre.
  • Clase (.clase): Reutilizable.
  • ID (#id): Único.

Selectores de Atributo

Seleccionan elementos que tienen un atributo específico o un valor determinado.

input[type="text"] { border: 1px solid; }

Selectores Descendientes

Selecciona un elemento que está dentro de otro, sin importar el nivel de profundidad.

div p { color: gray; }

Afecta a todos los párrafos que vivan dentro de un div.

Selectores de Hijo Directo (>)

A diferencia del descendiente, este SOLO selecciona a los hijos inmediatos (el primer nivel).

ul > li { list-style: none; }

Especificidad

Si un elemento tiene un ID, una Clase y una Etiqueta aplicándole el mismo color, el navegador prioriza el ID por ser el más específico.

Pseudo-clases (:hover)

Seleccionan elementos en base a su estado (como pasar el mouse por encima) o su posición en el árbol.

:hover
:nth-child(n)
:first-child

Pseudo-elementos (::before)

Permiten estilar una parte específica del elemento o insertar contenido visual desde CSS.

p::first-letter { font-size: 2em; }

Selectores Adyacentes (+)

Selecciona el elemento que está inmediatamente después de otro al mismo nivel.

h1 + p { margin-top: 0; }

Agrupación

Para aplicar el mismo estilo a varios elementos, se separan los selectores por coma.

h1, h2, h3 { font-family: Poppins; }

Combinadores

  • A B Descendiente
  • A > B Hijo directo
  • A + B Hermano adyacente

Efecto Cebra

tr:nth-child(even) {

  background: #f2f2f2;

}
