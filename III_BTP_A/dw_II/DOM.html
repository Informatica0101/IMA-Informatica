Dominio del DOM

El Document Object Model (DOM) es una interfaz de programación para documentos HTML y XML.

  • Representa la estructura del documento como un árbol de nodos.
  • Permite que programas (como JavaScript) accedan y manipulen el contenido, estructura y estilo de una página web.

Estructura del DOM (Árbol)

El DOM organiza los elementos de una página web en una estructura jerárquica, similar a un árbol.

  • Cada parte del documento (elementos, atributos, texto) es un nodo.
  • Los nodos tienen relaciones: padres, hijos, hermanos.

Tipos de Nodos Principales

El DOM se compone de diferentes tipos de nodos:

Nodo Documento

Representa todo el documento HTML (el objeto document). Es la raíz del árbol.

Selección de Elementos del DOM

Antes de poder manipular cualquier parte de una página web con JavaScript, primero debemos seleccionar los elementos específicos con los que queremos interactuar.

Existen varios métodos para hacer esto, y cada uno es útil en diferentes situaciones.

El Objeto Global document

En el contexto del DOM, document es el objeto global que representa toda la página web cargada en el navegador.

  • Es la puerta de entrada a todo el contenido de la página HTML.
  • Contiene métodos y propiedades para acceder, crear y modificar elementos, manejar eventos y mucho más.
  • Piensa en document como el "cerebro" o el "controlador principal" de tu página web.

document.getElementById('idElemento')

Este método es el más común para seleccionar un único elemento en tu HTML, siempre y cuando tenga un atributo id único.

  • getElementById: Es el método que busca un elemento por el valor de su atributo id.
  • El id debe ser único en todo el documento HTML.
  • Devuelve el elemento encontrado o null si no lo encuentra.

document.querySelector('selectorCSS')

Este método es muy versátil, ya que te permite seleccionar el primer elemento que coincida con un selector CSS específico.

  • querySelector: Busca un elemento usando cualquier selector CSS (clases, IDs, etiquetas, atributos, etc.).
  • Si hay múltiples coincidencias, solo devuelve la primera.
  • Devuelve el elemento o "null" si no lo encuentra.

document.querySelectorAll('selectorCSS')

Similar a querySelector, pero este método devuelve una NodeList (una colección de nodos) de todos los elementos que coinciden con el selector CSS.

  • querySelectorAll: Busca todos los elementos que cumplen con el selector.
  • La NodeList devuelta es similar a un array, permitiendo iterar sobre los elementos (ej. con forEach).
  • Si no encuentra coincidencias, devuelve una NodeList vacía.
