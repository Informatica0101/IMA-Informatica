Guía Completa de Propiedades CSS

Domina los estilos fundamentales para crear interfaces web profesionales y modernas.

1. Introducción a las Propiedades

Las propiedades CSS son los atributos que definen el aspecto visual de un elemento HTML. Cada propiedad acepta uno o más valores que determinan cómo se renderizará el elemento en el navegador.

selector {
  propiedad: valor;
}

2. Texto y Tipografía

Estas propiedades controlan la apariencia del contenido textual, desde el color y tamaño hasta el espaciado y estilo de la fuente.

Propiedad Descripción Valores Comunes
color Define el color del texto. nombre, hex, rgb()
font-size Ajusta el tamaño de la letra. px, rem, em, %
font-weight Define el grosor de la letra. normal, bold, 100-900
text-align Alinea el texto horizontalmente. left, center, right, justify

Visualización:

Este texto tiene color azul, es de 1.5rem y está centrado.

p {
  color: #1e40af;
  font-size: 1.5rem;
  font-weight: 800;
  text-align: center;
}

3. Modelo de Caja (Box Model)

El modelo de caja es la base del diseño web. Cada elemento es una caja rectangular compuesta por contenido, relleno (padding), borde y margen.

Propiedad Descripción Valores Comunes
width / height Ancho y alto del elemento. px, %, vh/vw, auto
padding Espacio interno (entre contenido y borde). px, rem
margin Espacio externo (entre elementos). px, auto
border Línea que rodea el padding. 1px solid black
box-sizing Cómo se calcula el tamaño total. border-box, content-box

Visualización:

Caja con Padding y Borde
div {
  width: 200px;
  padding: 20px;
  border: 4px solid #3b82f6;
  margin: 10px;
  background-color: #dbeafe;
}

4. Diseño Moderno (Flexbox y Grid)

Estas propiedades permiten crear layouts complejos y responsivos de forma mucho más sencilla que los métodos tradicionales.

Flexbox (Unidimensional)

Propiedad Descripción Valores Comunes
display: flex Activa el contenedor flexible. -
justify-content Alineación en el eje principal. center, space-between
align-items Alineación en el eje cruzado. center, stretch
flex-direction Dirección de los elementos. row, column

5. Posicionamiento y Efectos

Controlan dónde se ubican los elementos y cómo interactúan visualmente con el usuario a través de transparencias, sombras y transformaciones.

Propiedad Descripción Valores Comunes
position Tipo de posicionamiento del elemento. relative, absolute, fixed, sticky
opacity Nivel de transparencia. 0.0 a 1.0
box-shadow Añade sombra a la caja. x y blur color
cursor Cambia el puntero del ratón. pointer, not-allowed

Visualización:

Pásame el mouse (Sombra y Escala)
.tarjeta {
  background-color: white;
  padding: 20px;
  border-radius: 12px;
  box-shadow: 0 10px 25px rgba(0,0,0,0.2);
  cursor: pointer;
  transition: transform 0.3s;
}
.tarjeta:hover {
  transform: scale(1.1);
}
