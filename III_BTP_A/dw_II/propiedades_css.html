Diccionario de Propiedades CSS

1. Texto y Tipografía

Propiedades fundamentales para dar formato al contenido escrito de tu sitio web.

color, font-size, font-family

Define el color, tamaño y la familia de fuente del texto.

PropiedadDescripciónValores
colorColor del texto.blue, #ff0000, rgb()
font-sizeTamaño de la fuente.px, rem, em, %
font-familyTipo de letra.Arial, serif, sans-serif

Texto en rojo, 1.25rem y fuente monospace.

p { color: #ef4444; font-size: 1.25rem; font-family: monospace; }

font-weight, font-style, line-height, letter-spacing

Controla el grosor, estilo (itálica), altura de línea y espacio entre letras.

PropiedadDescripciónValores
font-weightGrosor de la letra.bold, normal, 100-900
font-styleEstilo de la fuente.normal, italic, oblique
line-heightEspacio entre líneas.1.5, 32px, 150%
letter-spacingEspacio entre caracteres.2px, 0.1em, normal

TEXTO EXPANDIDO E ITÁLICO

p { font-weight: 900; font-style: italic; line-height: 2; letter-spacing: 5px; }

text-align, text-decoration, text-transform, text-shadow

Alineación, decoraciones (subrayado), transformación (mayúsculas) y sombras.

PropiedadDescripciónValores
text-alignAlineación horizontal.left, center, right, justify
text-decorationLíneas decorativas.none, underline, line-through
text-transformCambio de mayúsculas.uppercase, lowercase, capitalize
text-shadowSombra del texto.2px 2px 4px #000

Título con Sombra y Subrayado

p { text-align: center; text-decoration: underline; text-transform: uppercase; text-shadow: 2px 2px 5px gray; }

2. Fondos y Colores

Propiedades para controlar el fondo de los elementos.

background-color, background-image, background-size

PropiedadDescripciónValores
background-colorColor de fondo.transparent, hex, rgba()
background-imageImagen de fondo.url('img.jpg'), linear-gradient()
background-sizeTamaño de la imagen.cover, contain, 100% 100%
Fondo con Degradado
div { background: linear-gradient(90deg, #3b82f6, #10b981); }

background-position, background-repeat

PropiedadDescripciónValores
background-positionPosición de la imagen.center, top right, 50% 50%
background-repeatRepetición de imagen.no-repeat, repeat-x, repeat-y
div { background-image: url('logo.png'); background-repeat: repeat-x; background-size: 50px; }

3. Modelo de Caja (Dimensiones y Espaciado)

Propiedades que definen el tamaño, bordes y márgenes de los elementos.

width, height, padding, margin

PropiedadDescripciónValores
width / heightAncho y alto.px, %, auto, 100vh
paddingEspacio interno.10px (todos), 5px 10px (v h)
marginEspacio externo.20px, auto (centra)
Caja con padding de 30px
div { padding: 30px; margin: 10px; background-color: #fef08a; border: 2px solid #eab308; }

border, border-radius, box-sizing, outline

PropiedadDescripciónValores
borderBorde del elemento.2px solid blue, dashed
border-radiusRedondeo de esquinas.8px, 50% (círculo)
box-sizingCálculo del tamaño.border-box, content-box
outlineLínea externa al borde.3px solid red
div { width: 80px; height: 80px; border-radius: 50%; border: 4px solid #1e3a8a; outline: 3px solid #93c5fd; }

4. Diseño Moderno (Flexbox y Grid)

Las herramientas más potentes para crear layouts dinámicos.

display, justify-content, align-items, flex-direction, flex-wrap

PropiedadDescripciónValores
display: flexActiva Flexbox.-
justify-contentAlineación horizontal.center, space-between, end
align-itemsAlineación vertical.center, stretch, start
flex-directionDirección de flujo.row, column
flex-wrapSalto de línea.wrap, nowrap
.container { display: flex; justify-content: space-between; align-items: center; }

display: grid, grid-template-columns/rows, gap, grid-column

PropiedadDescripciónValores
display: gridActiva Grid.-
grid-template-columnsColumnas.repeat(3, 1fr), 200px 1fr
gapEspacio entre celdas.10px, 1rem
grid-columnEspacio que ocupa.span 2, 1 / 3
Cabecera (span 2)
.grid { display: grid; grid-template-columns: 1fr 1fr; gap: 10px; }
.item { grid-column: span 2; }

5. Posicionamiento y Visibilidad

Determina cómo se ubican los elementos en el flujo del documento.

position, z-index, top/right/bottom/left

PropiedadDescripciónValores
positionTipo de ubicación.relative, absolute, fixed, sticky
z-indexOrden de apilado.1, 999, -1
top / bottom...Coordenadas.0, 50px, 10%
Z-index 2
Z-index 1
.caja { position: absolute; top: 10px; left: 10px; z-index: 2; }

overflow, visibility, opacity, object-fit

PropiedadDescripciónValores
overflowContenido excedente.hidden, scroll, auto
visibilityVisibilidad.visible, hidden
opacityTransparencia.0.0 (invisible) a 1.0
object-fitAjuste de imágenes.cover, contain, fill
img { width: 100px; height: 100px; object-fit: contain; opacity: 0.6; }

6. Efectos, Transformaciones e Interactividad

Propiedades para añadir dinamismo y acabados visuales.

box-shadow, transform, transition, filter, cursor

PropiedadDescripciónValores
box-shadowSombra de caja.0 4px 10px #000
transformModificar forma.rotate(45deg), scale(1.2)
transitionAnimación suave.all 0.3s ease
filterEfectos visuales.blur(5px), grayscale(1)
cursorTipo de puntero.pointer, crosshair, grab
HOVER ME
div {
  box-shadow: 0 10px 20px rgba(99, 102, 241, 0.4);
  transform: rotate(-5deg);
  transition: all 0.3s;
}
