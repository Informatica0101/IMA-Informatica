Propiedades de Texto

Permiten controlar la apariencia de las fuentes y el contenido escrito.
Ejemplos: color, font-family, font-size, line-height.

Propiedades de Fondo

Controlan el área detrás del contenido. Podemos usar colores, imágenes o degradados.

  • background-color: Color sólido.
  • background-image: Imagen de fondo.
  • background-size: Tamaño (cover, contain).

Propiedades de Caja (Box Model)

Todo en CSS es una caja. Las propiedades clave son padding (espacio interno), margin (espacio externo) y border.

Caja

Dimensiones: width y height

Establecen el ancho y alto de los elementos.
Se pueden usar valores fijos (px) o relativos (%, vh, vw).

Posicionamiento: position

  • static: Posición normal (por defecto).
  • relative: Relativo a su posición original.
  • absolute: Relativo al padre más cercano posicionado.
  • fixed: Fijo en la pantalla (no se mueve al hacer scroll).

Ejercicio: Estilo de Botón

Escribe el código CSS para un botón con:

  • Fondo azul
  • Texto blanco
  • Bordes redondeados (border-radius: 10px)
  • Espacio interno (padding: 15px)

Visibilidad: display

Define cómo se comporta el elemento en el flujo de la página.

  • block: Ocupa todo el ancho.
  • inline: Solo el espacio de su contenido.
  • none: Desaparece por completo.

Flexbox: display flex

La forma moderna de alinear elementos en filas o columnas de manera flexible y fácil.

Sombras: box-shadow

Agrega profundidad a las cajas. Sintaxis: h-offset v-offset blur spread color.

Elemento con Sombra

Transiciones: transition

Permite que los cambios de propiedad (como pasar el mouse por encima) sean suaves y no instantáneos.

transition: all 0.3s ease;

Filtros: filter

Aplica efectos visuales a imágenes o elementos (ej: grayscale, blur, brightness).

Reto de Código

.tarjeta {

  width: 300px;

  background: #fff;

  border: 1px solid #ddd;

  padding: 20px;

  margin: 10px;

}
