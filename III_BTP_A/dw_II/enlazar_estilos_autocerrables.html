El Método de Enlace Externo

En lugar de escribir estilos dentro de cada etiqueta, los centralizamos en un archivo .css.

¿Por qué es superior? Permite que un solo cambio en el archivo CSS actualice instantáneamente cientos de páginas HTML vinculadas.

Atributos del Elemento <link>

Esta etiqueta se ubica siempre dentro de la sección <head> y requiere tres parámetros:

  • rel="stylesheet": Define la relación entre el HTML y el archivo vinculado.
  • type="text/css": Indica el tipo de contenido que el navegador debe esperar.
  • href: La ruta de ubicación del archivo CSS.

Entendiendo el atributo 'href'

El valor de href depende de dónde esté guardado el archivo:

Ruta Relativa: href="estilos.css" (Si están en la misma carpeta).
Ruta Absoluta: href="https://sitio.com/estilos.css" (Carga desde internet).

Sintaxis Correcta

<head>
  <title>Mi Proyecto</title>
  <link rel="stylesheet"
        type="text/css"
        href="css/style.css" />
</head>

Nota: En este ejemplo, el archivo CSS está dentro de una carpeta llamada 'css'.

Ventaja: Código Limpio

Al separar la estructura (HTML) del diseño (CSS):

  • El archivo HTML es más legible y corto.
  • Varios diseñadores pueden trabajar en el mismo proyecto sin interferir.
  • Se reduce el peso de carga total del sitio web.

Etiquetas Autocerrables (Void Elements)

En HTML, la mayoría de las etiquetas vienen en pares (apertura y cierre). Sin embargo, hay excepciones.

Son etiquetas que no contienen texto u otros elementos dentro, por lo que no necesitan una etiqueta de cierre separada como </tag>.

La Barra Diagonal (/)

Aunque en HTML5 moderno la barra al final es opcional, se recomienda usarla por compatibilidad y claridad:

<img src="foto.jpg">

(Válido en HTML5)

<img src="foto.jpg" />

(Recomendado/Estándar)

Ejemplos Esenciales I

  • <br />: Inserta un salto de línea forzado.
  • <hr />: Crea una línea divisoria horizontal.
  • <img />: Inserta una imagen mediante el atributo src.

Ejemplos Esenciales II

  • <meta />: Define metadatos (como el conjunto de caracteres).
  • <input />: Crea campos de entrada en formularios.
  • <link />: Vincula recursos externos como hojas de estilo.

¿Qué NO hacer?

Intentar cerrar una etiqueta autocerrable con una etiqueta de cierre separada es un error de sintaxis:

Incorrecto:

<br> Contenido </br>

El navegador ignorará el cierre o podría renderizar mal la página.
