Mejor fuera de línea que en línea

Escribir el CSS en un archivo aparte es la mejor forma de darle estilo a tu sitio web.

Ventaja Clave: Con un único archivo CSS haces cambios en un solo lugar que afectan a todas las etiquetas de estilo de tu proyecto.

Problemas del estilo inline

Usar <p style="color:red"> no es eficiente porque:

  • Tienes que escribir el mismo código una y otra vez.
  • Si quieres cambiar el estilo de varios elementos, debes modificar cada etiqueta individualmente.
  • Mezcla la estructura (HTML) con el diseño (CSS).

Enlazar estilos CSS

Para que el HTML pueda "ver" la información de CSS, usamos la etiqueta <link> dentro de <head>.

La etiqueta necesita tres atributos fundamentales: type, rel y href.

Los 3 atributos de <link>

  1. type: Siempre debe ser igual a "text/css".
  2. rel: Siempre debe ser igual a "stylesheet".
  3. href: La dirección web (o ruta local) de tu archivo CSS.

Ejemplo de enlace

<link type="text/css"
      rel="stylesheet"
      href="stylesheet.css" />

Este código conecta tu archivo HTML con un archivo llamado "stylesheet.css".

Etiquetas Autocerrables

Son aquellas que no necesitan una etiqueta de cierre separada (como </p>) porque nunca contienen texto u otros elementos dentro.

Se cierran usando un solo par de < /> al final de la etiqueta de inicio.

Sintaxis de Autocierre

Se agrega una barra diagonal antes del signo "mayor que":

<link ... />
<img src="..." />

Esto le indica al navegador que la etiqueta ha terminado ahí mismo.

La etiqueta <img>

Es uno de los ejemplos más comunes de etiquetas autocerrables.

<img src="paisaje.jpg" />

No lleva </img> al final porque la imagen es el contenido en sí misma.

Etiquetas <br> y <hr>

Otras etiquetas que ahorran tiempo y esfuerzo al ser autocerrables:

  • <br />: Inserta un salto de línea simple.
  • <hr />: Dibuja una línea horizontal en la página.

Resumen de la Sesión

Enlace Externo: Permite separar el diseño de la estructura, facilitando el mantenimiento.
Autocierre: Simplifica la sintaxis del documento para elementos sin contenido interno.
