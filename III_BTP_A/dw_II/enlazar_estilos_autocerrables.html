Etiquetas con Contenido

La mayoría de etiquetas envuelven un contenido (texto, imágenes u otras etiquetas) y necesitan una apertura y un cierre.

<p> Contenido </p>

Etiquetas Autocerrables

Son aquellas que no contienen texto ni envuelven a otros elementos. Por lo tanto, NO tienen etiqueta de cierre. Representan una acción o una inserción simple.

Ejemplos Comunes

  • <br>: Salto de línea.
  • <hr>: Línea horizontal.
  • <img>: Imagen.
  • <input>: Campo de formulario.

La etiqueta <link>

Es una etiqueta autocerrable fundamental. Se usa para conectar el archivo HTML con recursos externos, principalmente archivos de estilo CSS.

<link rel="stylesheet" href="estilos.css">

Atributos de <link>

  • rel: Define la relación (siempre "stylesheet").
  • href: La ruta del archivo .css.
  • type: Tipo de archivo (opcional).

Rutas de Archivo

Si tu HTML está en la raíz y tu CSS está dentro de una carpeta llamada css, el href debe apuntar correctamente:

href="css/estilos.css"

Ventajas del CSS Externo

  • Reutilización: Un solo CSS para todo el sitio.
  • Orden: HTML para estructura, CSS para diseño.
  • Carga: El navegador guarda el archivo en caché.

Etiqueta <meta>

Otra etiqueta autocerrable vital. Define metadatos como el idioma, autor o la compatibilidad con dispositivos móviles.

<meta charset="UTF-8">

Sintaxis Limpia

En HTML5 actual, la barra al final es opcional y se prefiere no usarla para mantener el código más limpio.

Correcto: <br>
Antiguo: <br />

Buenas Prácticas

Mantén tus archivos organizados. Usa nombres claros como main.css o layout.css.

Estructura Head

<head>

  <meta charset="UTF-8">

  <title>Mi Web</title>

  <link rel="stylesheet" href="style.css">

</head>

Organización

"Separar el contenido de la presentación es la base del diseño web moderno y mantenible."

PRÁCTICA DE ENLACE

Crear un proyecto con un archivo index.html y un style.css en una carpeta separada. Enlazar ambos y cambiar el color de fondo del body.
