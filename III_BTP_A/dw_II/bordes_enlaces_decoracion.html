Bordes / Enlaces y text-decoration en CSS

Lic. Pablo Antonio Peña Mancia

Fecha: 24 de mayo de 2026

Grado: III BTP Informática Sección "A"

Propiedad Border

Muchos elementos HTML admiten la propiedad borde (border). Es especialmente útil para definir límites y organizar tablas.

La propiedad border requiere tres valores fundamentales: ancho, estilo y color.

Sintaxis del Borde

selector {
  border: 2px solid red;
}
  • 2px: El grosor de la línea.
  • solid: El estilo (liso, punteado, etc.).
  • red: El color del borde.

Tipos de Estilos Comunes

dashed (punteado)
dotted (puntos)
solid (liso)
double (doble)

Esquinas Redondeadas

Puedes conseguir bordes más redondos o rectos al modificar el valor de border-radius.

div {
  border: 2px solid blue;
  border-radius: 10px;
}

Bordes en Tablas

Para visualizar mejor una tabla, es común aplicar bordes a las celdas (td) y a la tabla misma.

Ejemplo: `td { border: 1px dashed blue; }` crea celdas con bordes azules punteados.

Ejercicio 1: Bordes

Crea un archivo HTML con una tabla de 3x3 y aplica estilos diferentes:

  • Tabla con borde sólido negro de 1px.
  • Celdas (td) con borde punteado azul.
  • Un encabezado (th) con texto en color rojo.

Personalización de Enlaces

Los enlaces (etiqueta <a>) tienen propiedades de texto normal, pero incluyen la propiedad especial text-decoration.

Por defecto, los enlaces son azules y están subrayados, pero esto se puede cambiar para personalizar el diseño.

Valores de Decoración

  • underline: Subrayado normal (por defecto).
  • overline: Línea sobre el texto.
  • line-through: Texto tachado.
  • none: Quita cualquier decoración.

Estados de un Enlace

Las pseudoclases permiten cambiar el estilo según la interacción del usuario:

  • a:link: Un enlace que aún no ha sido visitado.
  • a:visited: Un enlace que el usuario ya visitó.
  • a:hover: Cuando el usuario pasa el cursor por encima.

Botón con Enlace

Podemos quitar el subrayado y cambiar el color para que parezca un botón real:

a {
  color: #cc0000;
  text-decoration: none;
}
a:hover {
  font-weight: bold;
}

Foco en Elementos Específicos

¿Y si quieres enfocarte en las <p> que están dentro de varios <div>?

div div p {
  color: red;
}

Esto solo afectará a los párrafos anidados en dos capas de divs.

Ejercicio 2: Navegación

Diseña tres botones de navegación que cumplan lo siguiente:

  • Sin subrayado predeterminado.
  • Con bordes sólidos de 2px y esquinas redondeadas.
  • Que cambien de color al pasar el cursor (hover).
