Propiedad border

Define la línea que rodea al elemento. Se compone de tres valores:

  • Width: Grosor (ej: 2px).
  • Style: Estilo (solid, dashed, dotted).
  • Color: Color de la línea.
Borde Sólido

Redondeo: border-radius

Permite redondear las esquinas de una caja. Si aplicas el 50% a un elemento cuadrado, obtendrás un círculo perfecto.

Estilo de Enlaces

Los enlaces (<a>) tienen estados especiales que podemos estilar usando pseudo-clases.

  • :link: Estado normal.
  • :hover: Al pasar el mouse.
  • :active: Al hacer clic.
  • :visited: Ya visitado.

Quitar el Subrayado

Por defecto, los navegadores subrayan los enlaces. Usamos text-decoration: none para una apariencia más limpia y moderna.

Enlace sin subrayado

Listas y Viñetas

La propiedad list-style-type permite cambiar o quitar los puntos de las listas (ul) o números de las listas (ol).

list-style-type: none; (Quita viñetas)

Ejercicio: Botón Interactivo

Crea el código para un enlace que parezca un botón:

  • Borde de 1px sólido azul.
  • Sin subrayado.
  • Que cambie el color de fondo al pasar el mouse (:hover).
  • Esquinas redondeadas de 5px.

Contornos: outline

Es similar al borde, pero no ocupa espacio en el modelo de caja y se dibuja por fuera del borde. Muy usado para indicar el foco del teclado.

Cursores: cursor

Define qué icono de mouse se muestra al pasar sobre el elemento (pointer, wait, help, move, not-allowed).

Sombras: box-shadow

Agrega profundidad a las cajas. Sintaxis: horizontal, vertical, desenfoque, color.

Tarjeta con Sombra

Efectos de Transición

Permite que los cambios de estilo (como el color en hover) sean suaves.

transition: background 0.3s;

Pseudo-elementos

Permiten estilar partes específicas, como la primera letra de un párrafo (::first-letter) o agregar contenido antes de un elemento (::before).

Reto de Código

a:hover {

  color: white;

  background: #2563eb;

  border-radius: 20px;

  transition: 0.5s;

}
