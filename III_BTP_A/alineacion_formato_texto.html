Propiedad text-align

Define la alineación horizontal del texto dentro de un contenedor. Los valores principales son:

  • left: Por defecto, a la izquierda.
  • right: A la derecha.
  • center: Centrado.
  • justify: Justificado (bordes rectos).

Decoración: text-decoration

Se usa para agregar líneas al texto. Esencial para quitar el subrayado de los enlaces (valor none) o tachar precios.

  • underline (subrayado)
  • line-through (tachado)
  • overline (línea superior)

Transformación: text-transform

Permite controlar las mayúsculas de forma automática, sin cambiar el HTML original.

uppercase (MAYÚSCULAS)
capitalize (Iniciales)

Interlineado: line-height

Controla el espacio vertical entre las líneas de un párrafo. Un valor común es 1.5 o 1.6 para mejorar la legibilidad en pantallas.

Espaciado de Letras y Palabras

  • letter-spacing: Espacio entre caracteres (ej: 2px).
  • word-spacing: Espacio entre palabras completas.

Ejercicio: Estilo de Párrafo

Escribe el código CSS para un párrafo con:

  • Texto centrado
  • Todo en mayúsculas
  • Interlineado doble (2)
  • Color azul

Sombra de Texto: text-shadow

Agrega efectos de profundidad al texto. Sintaxis: horizontal vertical difuminado color.

TEXTO CON SOMBRA

Sangrías: text-indent

Define el espacio vacío al inicio de la primera línea de un párrafo. Muy usado en artículos académicos o literarios.

p { text-indent: 50px; }

Propiedad vertical-align

Define la alineación vertical de elementos en línea (inline) o celdas de tabla. Valores: top, middle, bottom.

Propiedad white-space

Controla cómo se maneja el espacio en blanco y los saltos de línea. El valor nowrap evita que el texto pase a la siguiente línea.

Dirección: direction

Se usa para idiomas que se leen de derecha a izquierda (RTL) como el árabe. Valor por defecto: ltr (izquierda a derecha).

Reto de Código

// Combinación de propiedades de formato

selector {

  text-align: justify;

  text-transform: lowercase;

  text-decoration: underline;

  line-height: 1.8;

}
