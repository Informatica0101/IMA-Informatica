Alineación de Texto

La propiedad CSS text-align se utiliza para establecer la alineación horizontal de un texto.

left (Izquierda)

Ejemplo: Atributo Style

<h1 style="text-align: center;">Título centrado</h1>
<p style="text-align: justify;">Texto justificado...</p>

Formato: Negrita e Itálica

Podemos usar etiquetas HTML o propiedades CSS:

Negrita

<b> o <strong>

CSS: font-weight: bold;

Decoración de Texto

Usamos la propiedad text-decoration.

<p style="text-decoration: underline;">Subrayado</p>
<p style="text-decoration: line-through;">Tachado</p>

Mayúsculas y Minúsculas

Interlineado (Line-height)

La propiedad line-height establece la distancia entre las líneas de un párrafo, mejorando la legibilidad.

style="line-height: 1.5;"

Etiquetas Semánticas de Texto

<mark>: Texto resaltado (amarillo).

Ejercicio: Perfeccionando el Estilo

Crea un archivo HTML y aplica:

  • Un título (h1) centrado y en mayúsculas.
  • Un párrafo justificado con un interlineado de 1.6.
  • Usa <mark> para resaltar una palabra importante.
  • Aplica subrayado a un enlace o frase clave.
