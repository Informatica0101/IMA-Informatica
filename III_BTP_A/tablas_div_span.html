¿Qué son las Tablas HTML?

Las tablas se utilizan para mostrar datos de forma organizada en filas y columnas. No deben usarse para crear el diseño general de la página.

  • 📊 Ideales para datos tabulares (listas de precios, horarios).
  • 🧱 Estructura rígida basada en celdas.

Estructura Básica

<table>
  <tr> (Fila)
    <th>Encabezado</th>
    <td>Dato</td> (Celda)
  </tr>
</table>

Contenedores Genéricos

Etiquetas que no tienen un significado semántico pero sirven para agrupar contenido.

<div>

Elemento de BLOQUE. Ocupa todo el ancho disponible.

<span>

Elemento en LÍNEA. Solo ocupa el espacio de su contenido.

Uso de <div>

Se usa para agrupar secciones grandes o crear bloques de diseño.

<div class="contenedor">
  <h2>Título</h2>
  <p>Contenido agrupado...</p>
</div>

Uso de <span>

Se usa para aplicar estilos a partes específicas de un texto.

<p>
  El cielo es <span style="color: blue;">azul</span>.
</p>
