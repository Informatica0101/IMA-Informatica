¿Qué es un Hipervínculo?

Es la esencia de la Web (World Wide Web). Permite conectar una página con otra, o con un recurso (imagen, PDF, video), creando una red de información interconectada.

La etiqueta <a> (Anchor)

Se usa para crear enlaces. El atributo más importante es href, que indica el destino del enlace.

<a href="https://google.com">Ir a Google</a>

Atributo target

Define dónde se abrirá el enlace:

  • _self: En la misma pestaña (por defecto).
  • _blank: En una pestaña nueva.

Enlaces Internos (Anclas)

Permiten saltar a una sección específica de la misma página usando el id del elemento.

<a href="#contacto">Ir a Contacto</a>

La etiqueta <img>

Se usa para insertar imágenes. Es una etiqueta autocerrable (no necesita </img>).

<img src="paisaje.jpg" alt="Montañas">

Ejercicio: Enlace con Imagen

Crea el código para que al hacer clic en una imagen de un logo, el usuario sea enviado a una red social en una pestaña nueva.

El atributo alt

Texto Alternativo: Es obligatorio por accesibilidad. Se muestra si la imagen falla al cargar y lo leen los lectores de pantalla para personas con discapacidad visual.

Atributos width y height

Definen el tamaño de la imagen en píxeles. Es mejor práctica controlar el tamaño desde CSS para mantener la responsividad.

<img src="..." width="200">

Formatos de Imagen

  • JPG: Fotos (pesan poco).
  • PNG: Logos con transparencia.
  • SVG: Gráficos vectoriales (no se pixela).
  • WebP: Formato moderno ultra optimizado.

Rutas Relativas vs Absolutas

Absoluta: URL completa (http://...).
Relativa: Archivos locales (imagenes/foto.jpg).

Favicon

Es la pequeña imagen que aparece en la pestaña del navegador. Se inserta en el head con la etiqueta <link>.

Reto de Código

<!-- Estructura de navegación -->

  <a href="index.html">Inicio</a> |

  <a href="servicios.html">Servicios</a> |

  <a href="mailto:info@ima.hn">Correo</a>

</nav>
