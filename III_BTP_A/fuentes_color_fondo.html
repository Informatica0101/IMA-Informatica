¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que usamos para darle estilo a nuestro HTML.

  • 🦴 HTML = Estructura (Huesos).
  • 💄 CSS = Presentación (Piel y maquillaje).
  • 🎨 Permite cambiar colores, fuentes, tamaños y mucho más.

Estilos en Línea

Es la forma más rápida de aplicar estilos directamente en una etiqueta HTML usando el atributo style.

<p style="propiedad: valor;">Texto</p>

Cambiando el color del texto

Usamos la propiedad color para cambiar el color de la letra.

<h1 style="color: blue;">Título Azul</h1>

Color de Fondo

Usamos background-color para el fondo de un elemento o de toda la página.

<body style="background-color: lightyellow;">

Cambiando el tipo de letra

Usamos font-family para elegir la fuente.

<p style="font-family: Arial;">Texto en Arial</p>

Tamaño de la fuente

Usamos font-size para definir qué tan grande es el texto.

<p style="font-size: 20px;">Texto grande</p>

Combinando varios estilos

Puedes poner muchas propiedades dentro del mismo atributo style separadas por punto y coma.

<p style="color: red; font-size: 18px; font-family: Verdana;">
  Este texto es rojo, de 18px y fuente Verdana.
</p>

Ejercicio: Mi página colorida

Abre tu archivo del CV anterior y aplica:

  • Cambia el color de fondo de la página.
  • Pon tu nombre en color azul y un tamaño de 30px.
  • Cambia la fuente de los párrafos a 'Arial' o 'Verdana'.
  • Resalta un título con un color de fondo diferente.
