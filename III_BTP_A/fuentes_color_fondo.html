Propiedad color

Define el color del texto. Podemos usar nombres (red), hexadecimal (#FF0000) o RGB (rgb(255,0,0)).

background-color

Define el color de fondo de un elemento. Se puede aplicar al body para toda la página o a bloques específicos (div, p) para resaltarlos.

Propiedad font-family

Define el tipo de letra. Siempre debemos incluir una "fuente de respaldo" por si el usuario no tiene instalada la principal.

font-family: 'Poppins', Arial, sans-serif;

Familias Genéricas

  • Serif (Con remates)
  • Sans-serif (Lisas)
  • Monospace (Ancho fijo)

Propiedad font-size

Define el tamaño del texto. Unidades comunes:

  • px: Píxeles (Tamaño fijo).
  • em / rem: Unidades relativas (Recomendado para móviles).
  • %: Porcentaje.

Ejercicio: Estilo de Título

Escribe el CSS para que un H1 tenga:

  • Color de fondo gris oscuro.
  • Texto de color blanco.
  • Fuente tipo Monospace.
  • Tamaño de 24 píxeles.

Propiedad font-weight

Define el grosor de la letra. Valores: normal, bold, lighter o números (100 a 900).

PESO 900

Propiedad font-style

Permite poner el texto en cursiva.

font-style: italic;

Este texto está en itálica.

Fuentes de Google (Google Fonts)

Permite usar miles de fuentes gratuitas cargándolas desde la nube. Solo necesitas copiar un enlace <link> en tu HTML.

Opacidad: opacity

Define la transparencia de un elemento y su contenido. Valores del 0.0 (invisible) al 1.0 (opaco).

50% Opacidad

Colores RGBA

A diferencia de opacity, rgba permite dar transparencia solo al fondo sin afectar al texto.

background-color: rgba(255, 0, 0, 0.5);

Atajo: font

Se pueden resumir todas las propiedades de fuente en una sola línea.

font: italic bold 20px Arial;
