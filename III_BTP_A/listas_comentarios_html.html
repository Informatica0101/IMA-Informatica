Listas No Ordenadas (ul)

Se usan para agrupar elementos que no requieren un orden específico. Por defecto, muestran viñetas (puntos).

<ul>
  <li>Elemento</li>
</ul>

Listas Ordenadas (ol)

Se usan cuando el orden es importante (instrucciones, rankings). Por defecto, muestran números.

<ol>
  <li>Primero</li>
  <li>Segundo</li>
</ol>

Atributo Type en ol

Puedes cambiar el estilo de numeración:

  • type="1" (Números)
  • type="A" (Letras Mayúsculas)
  • type="a" (Letras Minúsculas)
  • type="I" (Números Romanos)

Listas de Descripción (dl)

Ideales para glosarios o listas de términos y definiciones.

  • <dt>: Término.
  • <dd>: Definición.

Listas Anidadas

Es poner una lista dentro de otra. Muy común para menús desplegables.

<ul>
  <li>Frutas
    <ul> <li>Manzana</li> </ul>
  </li>
</ul>

Comentarios en HTML

Notas que el programador deja en el código. El navegador NO las muestra al usuario. Sirven para organizar el código.

<!-- Esto es un comentario -->

¿Para qué sirven?

  • Desactivar código temporalmente.
  • Marcar el inicio/fin de secciones.
  • Explicar lógica compleja.

Atajo de Teclado

En la mayoría de editores (VS Code, Sublime), puedes comentar rápido con:

Ctrl + /

El Atributo start

Permite que una lista ordenada no empiece en 1.

<ol start="50"> ... </ol>

El primer elemento será el número 50.

Atributo reversed

Invierte el orden de la numeración de mayor a menor.

<ol reversed> ... </ol>

</ul>
