Listas y Comentarios en HTML

Semana 4: Organizando el Contenido

Lic. Pablo Antonio Peña Mancia

Fecha: 01 de marzo de 2026

Grado: Duodécimo de Informática - Sección "A"

¿Para qué sirven las listas?

Las listas nos permiten organizar información relacionada de manera estructurada. Son fundamentales para:

  • 🗺️ Crear menús de navegación.
  • 📝 Enumerar pasos de una receta o proceso.
  • 🛒 Enlistar productos o características.

Listas Desordenadas: <ul>

Se usan cuando el orden de los elementos no importa.

<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>

Por defecto, muestran viñetas (puntos).

Listas Ordenadas: <ol>

Se usan cuando el orden es importante (pasos, rankings).

<ol>
  <li>Primer paso</li>
  <li>Segundo paso</li>
</ol>

Muestran números por defecto.

El ítem de lista: <li>

La etiqueta <li> (List Item) siempre debe estar dentro de un <ul> o un <ol>.

⚠️ ¡Nunca coloques texto directamente dentro de <ul> sin usar <li>!

Comentarios en HTML

Sirven para dejar notas al programador. El navegador los ignora.

<!-- Este es un comentario en HTML -->

Uso: Explicar partes complejas del código o "apagar" código temporalmente sin borrarlo.

Otros tipos de Comentarios

CSS / JS (Bloque)

/* Comentario de bloque */

JavaScript (Línea)

// Comentario de una línea

Es importante conocer la diferencia según el lenguaje.

Listas Anidadas

Puedes poner una lista dentro de otra.

<ul>
  <li>Frutas
    <ul>
      <li>Manzana</li>
    </ul>
  </li>
</ul>

Mantenibilidad

Usar comentarios y etiquetas de estructura adecuadas (listas) hace que tu código sea fácil de leer para otros programadores (o para ti mismo en el futuro).

"Escribe código como si la persona que lo fuera a mantener fuera un psicópata que sabe dónde vives."
