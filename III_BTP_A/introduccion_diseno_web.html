¿Por qué aprender HTML?

El World Wide Web (WWW) contiene información casi infinita. Las páginas web son la unidad básica donde está almacenada esta información.

Características de las páginas web:

  • • Contienen texto, imágenes, animaciones, sonido y video
  • • Son hipertexto: están unidas entre sí mediante enlaces
  • • Cada página web que ves está escrita en HTML
  • • HTML es como un esqueleto que da estructura a cada página

"HTML es el lenguaje de marcado estándar para crear páginas web."

Características del lenguaje HTML

Cuando se diseñó el WWW, se decidió que debía cumplir estas características:

Distribuido

Información repartida en páginas no muy grandes enlazadas entre sí.

Hipertexto

Debía ser fácil navegar por él mediante enlaces.

Compatible

Con todo tipo de computadoras (PCs, Macintosh, estaciones de trabajo...) y sistemas operativos.

Dinámico

El proceso de cambiar y actualizar la información debía ser ágil y rápido.

Como respuesta a estos requisitos se creó el HTML (HyperText Markup Language) - "Lenguaje hipertexto de marcas".

¿Qué es HTML?

HTML significa: Hyper Text Markup Language

En español: Lenguaje de Marcado de Hipertexto

Hipertexto

"Texto que contiene enlaces". Cada vez que haces clic en una palabra que te lleva a otra página, estás haciendo clic en un hipertexto.

Lenguaje de marcado

Lenguaje usado para que el texto haga algo más que aparecer en una página: puede convertir texto en imágenes, enlaces, cuadros, listas, y más.

HTML convierte un archivo de texto en una página web con diferentes tipos y tamaños de letra, imágenes, animaciones, formularios interactivos, etc.

Estructura básica de HTML

Configurando el esqueleto de la página: 

<!DOCTYPE html>
<html>
    <head>
        <title>Título de la página</title>
    </head>
    <body>
        Contenido visible de la página
    </body>
</html>

Terminología básica: Etiquetas

Para aprender HTML, debemos aprender cómo hablar sobre HTML:

Los símbolos < > se usan mucho en HTML

El texto dentro de < > se llama etiqueta.

Etiqueta de inicio

<html>

Marca el comienzo de un elemento

Etiqueta de cierre

</html>

Marca el final de un elemento

Las etiquetas casi siempre vienen de a pares: una etiqueta de inicio y una etiqueta de cierre.

Anidamiento de etiquetas

Las etiquetas pueden anidarse (colocarse unas dentro de otras): 

<primer_a_etiqueta>
    <segunda_etiqueta>
        Algún texto
    </segunda_etiqueta>
</primer_a_etiqueta>

Imagina que las etiquetas son como paréntesis

Cuando abres uno, después tienes que cerrarlo. Las etiquetas también pueden anidarse, así que hay que cerrarlas en el orden correcto.

Regla importante:

La última etiqueta que abres debe ser la primera que cierres.

El Encabezado (<head>)

El archivo HTML siempre consiste de dos partes: el encabezado y el cuerpo. 

<html>
    <head>
        <title>Mi página web</title>
    </head>
    <body>
        Contenido visible
    </body>
</html>

El encabezado <head> contiene:

  • • Información sobre el archivo HTML
  • • El título de la página (lo que se ve en la pestaña del navegador)
  • • Metadatos (información sobre la página)
  • • Enlaces a hojas de estilo y scripts

Todo dentro de nuestro archivo HTML irá en medio de las etiquetas de inicio <html> y de cierre </html>.

El Título (<title>)

El título es lo que vemos en la barra de títulos del navegador o pestaña de la página. 

<head>
    <title>Mi página web</title>
</head>

Cómo agregar un título a tu página:

  1. Agrega una etiqueta de inicio <head> y una de cierre </head>
  2. En medio, coloca una etiqueta <title> de inicio y una de cierre </title>
  3. Entre las etiquetas <title>, escribe un título para tu página web
  4. Guarda tu trabajo

El Cuerpo (<body>) y Párrafos

El cuerpo es donde se ingresa el contenido que se verá en la página real. 

<body>
    <p>¡Hola mundo!</p>
    <p>Este es mi primer párrafo en HTML.</p>
</body>

Etiqueta <p> para párrafos:

  • Cada párrafo empieza con <p> y termina con </p>
  • Podemos ingresar contenido entre las etiquetas
  • El cuerpo va después del encabezado

Encabezados (h1 a h6)

HTML permite tener más de un tamaño de encabezados para organizar el contenido. 

<body>
    <h1>Título principal (más grande)</h1>
    <h2>Subtítulo</h2>
    <h3>Encabezado nivel 3</h3>
    <h4>Encabezado nivel 4</h4>
    <h5>Encabezado nivel 5</h5>
    <h6>Encabezado nivel 6 (más pequeño)</h6>
</body>

Hay seis tamaños de encabezados: <h1> es el más grande y <h6> es el más pequeño.

Ejemplo completo de página HTML

<!DOCTYPE html>
<html>
<head>
    <title>Mi primera página web</title>
</head>
<body>
    <h1>Bienvenidos a mi sitio</h1>
    <p>Esta es mi primera página web creada con HTML.</p>
    <h2>Acerca de mí</h2>
    <p>Estoy aprendiendo diseño web.</p>
</body>
</html>

Pasos para crear y ver tu página:

  1. Abre un editor de texto (Bloc de notas)
  2. Copia el código anterior
  3. Guarda el archivo como mi_pagina.html
  4. Abre el archivo en tu navegador web
  5. ¡Verás tu primera página web!

Resumen - Introducción a HTML

  • ✅ HTML es el lenguaje estándar para crear páginas web
  • ✅ Significa HyperText Markup Language
  • ✅ Es distribuido, hipertexto, compatible y dinámico
  • ✅ No necesita programas especiales para crearse
  • ✅ Usa etiquetas entre < >
  • ✅ Tiene estructura básica: DOCTYPE, html, head, body
  • ✅ <title> define el título en la pestaña
  • ✅ <h1> a <h6> son encabezados
  • ✅ <p> define párrafos de texto
