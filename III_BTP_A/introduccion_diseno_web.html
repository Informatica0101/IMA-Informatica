1.2 Características del lenguaje HTML

  • Distribuido: Información repartida en páginas enlazadas entre sí.
  • Hipertexto: Debe ser fácil navegar por él.
  • Compatible: Con todo tipo de computadoras y sistemas operativos.
  • Dinámico: El proceso de cambiar y actualizar la información debe ser ágil.

Qué se necesita

No es necesario ningún programa especial. El código HTML no es más que texto.

Editores Básicos

Bloc de notas (Windows), TextEdit (Mac), WordPad.

Editores Dedicados

Visual Studio Express, VS Code, Sublime Text.

Recuerda guardar siempre con extensión .html

2. HTML y CSS

HTML

El esqueleto que le da estructura a cada página web.

CSS

La "piel y maquillaje" que cubre los huesos. Se encarga del diseño visual.

Estructura Básica

Configurando el esqueleto de la página:

<!DOCTYPE html>

<html>

...

</html>

  • <!DOCTYPE html>: Dice al navegador el lenguaje.
  • <html>: Comienza el documento.
  • </html>: Finaliza el documento.

2.1 Terminología básica: Etiquetas

El texto dentro de los símbolos < > se llama etiqueta.

Etiqueta de inicio

<html>

Etiqueta de cierre

</html>

Las etiquetas casi siempre vienen de a pares.

Anidamiento de etiquetas

Las etiquetas pueden estar unas dentro de otras, como paréntesis.

<padre>
  <hijo>Contenido</hijo>
</padre>

Importante: Se cierran en el orden inverso al que se abrieron.

El Encabezado (<head>)

Contiene información sobre el archivo HTML que no se ve directamente en la página, como el título y metadatos.

Se ubica entre las etiquetas <html> y antes de <body>.

El Título (<title>)

Es lo que vemos en la barra de títulos del buscador o en la pestaña de la página.

<head>

<title>Mi página web</title>

</head>

El Cuerpo (<body>)

Aquí es donde se ingresa el contenido: textos, imágenes y enlaces. Es lo que el usuario verá realmente.

Va inmediatamente después de cerrar el encabezado </head>.

2.1 Párrafos (<p>)

Cada párrafo empieza con una etiqueta <p> y termina con </p>.

<p>¡Hola mundo!</p>

¡Hola mundo!

2.2 Encabezados (h1 a h6)

HTML permite tener 6 tamaños de encabezados. <h1> es el más grande y <h6> el más pequeño.

Ejemplo de Código

<body>
  <h1>Mi Título</h1>
  <p>Primer párrafo.</p>
  <h3>Subtítulo</h3>
  <p>Segundo párrafo.</p>
</body>

Salto de línea y Línea Horizontal

Salto de línea (<br/>)

Se utiliza para crear espacios entre el texto, como dar Enter.

Línea Horizontal (<hr/>)

Se utiliza para separar secciones con una línea visual.

2.3 Hipervínculos (<a>)

Para mandar al usuario a otra página usamos vínculos. El atributo href indica el destino.

<a href="http://www.google.com">Buscador</a>
Este es un ejemplo de enlace

2.4 Agregar imágenes (<img>)

Usamos la etiqueta <img> con el atributo src para indicarle dónde buscar la imagen. No tiene etiqueta de cierre.

<img src="https://upload.wikimedia.org/.../Bandera_Honduras.png" />

3. Imágenes como Enlaces

Podemos convertir una imagen en un enlace anidando la etiqueta <img> dentro de una etiqueta <a>. 

<a href="http://www.honduras.com">
  <img src="bandera.png" />
</a>

¡Cuestionario de Diseño Web!

Resumen

  • 1. HTML se usa para darle estructura a los sitios web.
  • 2. Los archivos HTML tienen un encabezado <head> y un cuerpo <body>.
  • 3. Usamos <title> para el nombre de la página.
  • 4. Las etiquetas de encabezado (h1-h6) y párrafos (p) organizan el texto.
  • 5. Los enlaces (a) e imágenes (img) añaden interactividad y contenido visual.
