Introducción al Diseño Web

Conceptos básicos y Lenguaje HTML

Lic. Pablo Antonio Peña Mancia

Fecha: 1 de febrero de 2026

Grado: Duodécimo de Informática

1. ¿Qué es el Diseño Web?

Es la planificación, creación y mantenimiento de sitios web. Combina aspectos visuales (estética) con aspectos funcionales (usabilidad y arquitectura de información).

2. Evolución de la Web

Web 1.0

Estática, solo lectura. Páginas simples de texto e imágenes.

Web 2.0

Social y colaborativa. El usuario crea contenido (Blogs, RRSS).

Web 3.0

Semántica e inteligente. IA y descentralización.

3. Cliente y Servidor

La web funciona mediante una arquitectura de peticiones y respuestas.

  • Cliente: El navegador del usuario que solicita información (Chrome, Safari).
  • Servidor: Computadora remota que almacena los archivos y los envía al cliente.

4. Navegadores Web

Es el software encargado de interpretar el código HTML, CSS y JS para mostrarlo visualmente al usuario.

Chrome, Firefox, Edge, Safari...

5. Protocolos HTTP y HTTPS

Reglas para transferir datos en la web. La "S" en HTTPS significa "Seguro", indicando que los datos viajan cifrados.

Siempre utiliza HTTPS para proteger la privacidad del usuario.

6. Dominios y Hosting

Dominio

El nombre único del sitio web (Ej: google.com). Es la "dirección" de la casa.

Hosting

El espacio alquilado en el servidor para guardar los archivos. Es el "terreno" donde se construye la casa.

7. ¿Qué es HTML?

HyperText Markup Language. Es el lenguaje base de la web. Se utiliza para definir la estructura y el contenido de las páginas mediante etiquetas.

8. Estructura Básica

<!DOCTYPE html>

<html>

<head> ... </head>

<body> ... </body>

</html>

Todo documento HTML5 debe comenzar con la declaración doctype.

9. El elemento <head>

Contiene metadatos e información no visible directamente para el usuario, como el título de la pestaña, enlaces a estilos y scripts.

  • <title>: Título de la página.
  • <meta>: Codificación y descripción.
  • <link>: Conexión con CSS.

10. El elemento <body>

Contiene todo el contenido visible de la página web: textos, imágenes, videos, formularios y botones.

11. Títulos y Párrafos

<h1> al <h6>: Definen niveles de jerarquía en los encabezados.

<p>: Se utiliza para agrupar bloques de texto.

Título H1

Esto es un párrafo de ejemplo.

12. Enlaces (Etiqueta <a>)

Permiten navegar entre diferentes páginas. El atributo más importante es href (Hypertext Reference).

<a href="pagina.html">Ir a...</a>

13. Imágenes (Etiqueta <img>)

Es una etiqueta de auto-cierre. Requiere los atributos src (ruta) y alt (texto alternativo).

<img src="foto.jpg" alt="Descripción de la foto">

14. Listas en HTML

Desordenadas (<ul>)

  • Item A
  • Item B

Ordenadas (<ol>)

  1. Primero
  2. Segundo

15. Introducción a CSS

Cascading Style Sheets. Se encarga de la presentación visual del sitio (colores, fuentes, márgenes).

16. Selectores Básicos

Permiten elegir a qué elementos HTML aplicarles estilo.

Elemento: p { color: red; }

Clase: .mi-clase { ... }

ID: #mi-id { ... }
