Fecha: 1 de febrero de 2026

Grado: Duodécimo de Informática

¿Qué es el Diseño Web?

Es el proceso de planificar, conceptualizar y organizar contenido para internet. Implica la combinación de estética visual, usabilidad y funcionalidad técnica.

Evolución de la Web

Web 1.0

Web estática, de solo lectura. El usuario es un espectador pasivo.

Web 2.0

Web social y dinámica. Los usuarios crean contenido (Blogs, RRSS).

Web 3.0

Web semántica e inteligente. IA, descentralización y personalización.

Conceptos y Definiciones

  • Navegador: Aplicación para interpretar código web (Ej: Chrome, Firefox).
  • Servidor: Computadora que almacena y entrega los archivos de la web.
  • URL: Localizador de recursos uniforme (la dirección de la página).
  • Dominio: Nombre amigable asociado a una dirección IP (Ej: google.com).

Protocolos y Hosting

HTTP / HTTPS

Protocolo de transferencia de datos. La "S" indica que la conexión está cifrada y es segura.

Hosting (Alojamiento)

Servicio de alquiler de espacio en un servidor para que tu sitio sea accesible desde internet.

¿Qué es HTML?

HyperText Markup Language. Es el lenguaje estándar para crear la estructura de las páginas web. No es un lenguaje de programación, sino de marcado.

<p>Hola, esto es un párrafo en HTML</p>

Estructura Básica

<!DOCTYPE html>

<html lang="es">

<head>

<title>Mi Primera Web</title>

</head>

<body>

<h1>Bienvenidos</h1>

<p>Contenido visible aquí.</p>

</body>

</html>

Etiquetas Fundamentales

<h1> a <h6>

Encabezados de diferentes niveles.

<p>

Define un párrafo de texto.

<a href="...">

Crea enlaces a otras páginas.

<img src="...">

Inserta una imagen en la página.

Listas y Contenedores

Listas

  • <ul>: Desordenada.
  • <ol>: Ordenada (1, 2, 3).
  • <li>: Elemento de lista.

Contenedores

  • <div>: Bloque genérico.
  • <span>: Línea genérica.
  • <header> / <footer>

Conclusión
