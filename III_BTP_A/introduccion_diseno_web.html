¿Por qué aprender HTML?

El World Wide Web (WWW) contiene información casi infinita. Las páginas web son la unidad básica donde está almacenada esta información.

Características de las páginas web:

  • • Contienen texto, imágenes, animaciones, sonido y video
  • • Son hipertexto: están unidas entre sí mediante enlaces
  • • Cada página web que ves está escrita en HTML
  • • HTML es como un esqueleto que da estructura a cada página

Características del lenguaje HTML

Cuando se diseñó el WWW, se decidió que debía cumplir estas características:

Distribuido

Información repartida en páginas no muy grandes enlazadas entre sí.

¿Qué es HTML?

HTML significa: Hyper Text Markup Language

En español: Lenguaje de Marcado de Hipertexto

¿Qué se necesita para crear una página web?

Una de las características más importantes para el programador:

¡NO ES NECESARIO NINGÚN PROGRAMA ESPECIAL!

El código HTML no es más que texto. Lo único necesario para escribirlo es un editor de texto.

Estructura básica de HTML

Configurando el esqueleto de la página: 

<!DOCTYPE html>
<html>
    <head>
        <title>Título de la página</title>
    </head>
    <body>
        Contenido visible de la página
    </body>
</html>

Terminología básica: Etiquetas

Para aprender HTML, debemos aprender cómo hablar sobre HTML:

Los símbolos < > se usan mucho en HTML

El texto dentro de < > se llama etiqueta.

Anidamiento de etiquetas

Las etiquetas pueden anidarse (colocarse unas dentro de otras): 

<primer_a_etiqueta>
    <segunda_etiqueta>
        Algún texto
    </segunda_etiqueta>
</primer_a_etiqueta>

El Encabezado (<head>)

El archivo HTML siempre consiste de dos partes: el encabezado y el cuerpo. 

<html>
    <head>
        <title>Mi página web</title>
    </head>
    <body>
        Contenido visible
    </body>
</html>
