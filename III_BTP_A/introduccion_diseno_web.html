¿Qué es el Diseño Web?

Es una actividad que consiste en la planificación, diseño, implementación y mantenimiento de sitios web. Abarca aspectos como el diseño gráfico, la experiencia de usuario (UX) y la programación.

La Web: WWW

La World Wide Web es un sistema de documentos de hipertexto interconectados a los que se accede a través de Internet usando un navegador.

Trilogía del Diseño Web

  • HTML: Estructura y Contenido.
  • CSS: Presentación y Diseño.
  • JavaScript: Comportamiento e Interactividad.

El Navegador (Browser)

Es el software que interpreta el código (HTML/CSS/JS) y lo convierte en la interfaz visual que vemos. Ejemplos: Chrome, Firefox, Safari, Edge.

Servidor y Cliente

  • Cliente: Tu computadora o móvil que solicita la web.
  • Servidor: La computadora remota donde están guardados los archivos.

Ejercicio: El Proceso

Imagina que quieres crear una web para una pastelería. ¿Qué contenido iría en el HTML y qué colores/estilos usarías en el CSS?

Responsive Design

Es la técnica de diseño que asegura que una web se vea bien en cualquier dispositivo, desde una pantalla de cine hasta un smartwatch.

Experiencia de Usuario (UX)

Se enfoca en cómo se siente el usuario al navegar. Debe ser fácil, intuitivo y rápido. Una web que tarda mucho en cargar tiene mala UX.

Interfaz de Usuario (UI)

Se enfoca en la parte visual: los botones, los íconos, el espaciado y la combinación de colores que el usuario "toca".

Protocolo HTTP / HTTPS

Es el lenguaje de comunicación entre cliente y servidor. La "S" final significa Secure (Seguro) e indica que los datos van cifrados.

Editor de Código

Nuestra herramienta principal. El estándar actual es Visual Studio Code por su potencia y extensiones.

Reto de Código

</body>

</html>
