¿Qué es HTML?

HyperText Markup Language: No es un lenguaje de programación, es un lenguaje de marcado que define la estructura y el contenido de una página web mediante etiquetas.

Etiqueta (Tag)

Son los ladrillos de HTML. Van entre paréntesis angulares. La mayoría tiene una etiqueta de apertura y otra de cierre.

<p> Hola Mundo </p>

Atributo

Proporcionan información adicional sobre un elemento. Siempre van dentro de la etiqueta de apertura.

<img src="foto.jpg" alt="Descripción">

Aquí 'src' y 'alt' son los atributos.

¿Qué es CSS?

Cascading Style Sheets: Es el lenguaje que usamos para definir el diseño visual (colores, fuentes, posiciones) de los elementos HTML.

Selector CSS

Es el nombre que indica a qué elemento HTML queremos aplicarle el estilo.

h1 { color: blue; }

En este caso, 'h1' es el selector.

Ejercicio: Identificación

En el siguiente código: <a href="ima.hn">IMA</a>

  • ¿Cuál es la etiqueta?
  • ¿Cuál es el atributo?
  • ¿Cuál es el valor del atributo?

Declaración CSS

Es el conjunto de Propiedad y Valor. Van dentro de llaves {}.

Propiedad: color
Valor: red

DOM (Document Object Model)

Es la estructura de árbol que el navegador crea al leer el HTML. Permite que lenguajes como JavaScript interactúen con la página.

Semántica

Usar las etiquetas correctas para el contenido correcto (ej: <nav> para menús, <footer> para el pie). Ayuda al SEO y a la accesibilidad.

Responsive Design

Técnica de diseño que permite que una página web se adapte automáticamente al tamaño de pantalla del dispositivo (Móvil, Tablet, PC).

Hosting y Dominio

  • Dominio: El nombre de la web (ej: google.com).
  • Hosting: El servidor donde viven los archivos.

Reto de Código

<!DOCTYPE html>

<html>

<head> <title>Mi Web</title> </head>

<body>

  <h1>Bienvenida</h1>

  <p>Contenido principal</p>

</body>

</html>
