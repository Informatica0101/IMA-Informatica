Poder Analítico

Microsoft Excel no es solo una tabla de datos; es un motor de análisis masivo. Permite automatizar procesos de negocio y generar reportes inteligentes.

Fórmulas vs Funciones

Fórmulas

Creadas por el usuario usando operadores lógicos (+, -, *, /).

Funciones

Fórmulas predefinidas integradas en Excel (BUSCARV, SI, SUMA).

Funciones Lógicas (SI)

Permiten que la hoja de cálculo tome "decisiones" basadas en una condición.

=SI(A1>=70, "Aprobado", "Reprobado")

Búsqueda de Datos (BUSCARV)

Una de las funciones más potentes. Busca un valor en una columna y devuelve el dato de otra columna en la misma fila.

Fórmulas Matriciales

Realizan múltiples cálculos en uno o más conjuntos de valores y devuelven un solo resultado o múltiples resultados.

Ejercicio Práctico 1

En una hoja de Excel, escribe la función SI necesaria para mostrar "Beca" si el promedio de un alumno (B2) es mayor o igual a 90.

Anidamiento

Consiste en usar una función dentro de otra.
Ejemplo: Un SI dentro de otro SI para evaluar múltiples rangos de notas.

Validación de Datos

Restringe qué datos puede ingresar el usuario en una celda (ej. solo números, lista desplegable) para evitar errores humanos.

Formatos de Número

  • Moneda: Agrega símbolo L. y decimales.
  • Fecha: Formatea números como fechas reales.
  • Porcentaje: Multiplica por 100 y agrega %.

Auditoría de Fórmulas

Herramientas para rastrear precedentes y dependientes, permitiendo encontrar el origen de un error de cálculo.

El Corazón del Negocio

"Las funciones avanzadas de Excel transforman datos en bruto en conocimiento accionable para la toma de decisiones."

Lógica de Función

FUNCTION BUSCAR_DATO(valor, matriz, col)

  FOR EACH fila IN matriz

    IF fila[0] == valor THEN

      RETURN fila[col]

    ENDIF

  ENDFOR

ENDFUNCTION
