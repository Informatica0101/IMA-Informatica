¿Qué es VBA?

VBA (Visual Basic for Applications) es el lenguaje de programación de Microsoft que permite automatizar tareas en Excel y otros programas de Office.

Es la herramienta definitiva para crear aplicaciones personalizadas dentro de Excel.

El Editor de VBA

Para acceder, usamos el atajo ALT + F11. Es un entorno separado de la hoja de cálculo donde escribimos nuestros procedimientos (Macros).

Conceptos Clave

Módulo: El archivo donde se guarda el código.
Procedimiento (Sub): Un bloque de código de acción.
Objeto: Elementos de Excel (Celda, Rango).
Propiedad: Características (Valor, Color).

La Jerarquía de Objetos

VBA funciona de lo general a lo específico:

Application.Workbooks("L1").Worksheets("H1").Range("A1").Value = 10

Variables en VBA

Se declaran usando la palabra reservada Dim:

Dim nombre As String

Dim edad As Integer

Dim total As Double

Estructura de una Macro

Sub MiPrimeraMacro()

  // Instrucciones aquí

  Range("A1").Value = "IMA"

End Sub

MsgBox e InputBox

MsgBox

Muestra un mensaje al usuario.

InputBox

Pide un dato mediante una ventana.

Estructura Condicional

If Range("A1").Value > 70 Then

  MsgBox "Aprobado"

Else

  MsgBox "Reprobado"

End If

Archivos Habilitados

Para que el código funcione al cerrar y abrir el archivo, debes guardarlo con la extensión .xlsm.

El Poder de Automatizar

"VBA permite que Excel haga en un segundo lo que a un humano le tomaría todo el día."

Uso de Celdas

Dim i As Integer

For i = 1 To 10

  Cells(i, 1).Value = i * 2

Next i
