¿Qué es la Validación de Datos?

Es una herramienta de Excel que permite controlar qué valores pueden ingresar los usuarios en una celda. Sirve para evitar errores humanos y mantener la integridad de la información.

Importancia

  • Evita datos incoherentes (ej. edades negativas).
  • Estandariza entradas (ej. nombres de ciudades).
  • Guía al usuario con mensajes de entrada.

Tipos de Validación: Listas

Muestra un menú desplegable con opciones predefinidas.
Es el tipo más común para campos como "Género", "Día" o "Sede".

Opción 1 ▼

Tipos de Validación: Números

Permite restringir la entrada a Números Enteros o Decimales dentro de un rango (ej. Notas del 0 al 100).

0 < x < 100

Tipos de Validación: Fechas

Asegura que el dato ingresado sea una fecha válida y, opcionalmente, dentro de un periodo específico (ej. fechas de este año únicamente).

Longitud de Texto

Controla cuántos caracteres puede tener una celda. Útil para Identidades, Números de Teléfono o Códigos Postales.

Ejemplo: Identidad debe tener exactamente 13 dígitos.

Mensaje de Entrada

Aparece cuando el usuario selecciona la celda, dándole instrucciones antes de que escriba nada.

Instrucción: Ingrese solo números del 1 al 10.

Mensaje de Error

Es la alerta que salta cuando el usuario intenta ingresar un dato que NO cumple las reglas. Existen 3 estilos:

  • Detener: Bloquea la entrada (Obligatorio).
  • Advertencia: Avisa pero permite seguir.
  • Información: Solo comunica.

Ubicación en el Menú

Pestaña Datos → Grupo Herramientas de Datos → Validación de Datos.

Círculos de Validación

Excel puede marcar con un círculo rojo los datos que ya estaban en la hoja y que NO cumplen con la nueva regla de validación aplicada.

!

Listas Dependientes

Es una técnica avanzada donde las opciones de una lista cambian según lo que se seleccionó en otra lista (ej. Seleccionas "País" y la segunda lista solo muestra "Ciudades" de ese país).

Limpiando Validaciones

Para quitar todas las restricciones de una celda, debes entrar al menú de Validación y presionar el botón "Borrar todos".
