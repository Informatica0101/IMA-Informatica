La Importancia de la Calidad de Datos

En una hoja de cálculo, la toma de decisiones depende de la exactitud de los datos. Las herramientas de validación y filtrado nos ayudan a mantener la integridad y consistencia de la información.

Fundamentos: Integridad y Consistencia

Estas dos cualidades son el pilar de cualquier base de datos bien estructurada.

Integridad

Asegurar que los datos sean correctos, completos y que no se hayan modificado o dañado accidentalmente. La Validación ayuda a garantizarla.

1. Herramienta de Validación de Datos

La Validación de Datos es una función que permite restringir el tipo de información o los valores que los usuarios pueden introducir en una celda.

Validación de datos

Criterios Comunes de Validación

  • Lista (Desplegable): Permite elegir valores de una lista predefinida (ej. Nombres de departamentos, Sí/No).
  • Número Entero/Decimal: Restringe la entrada solo a números, permitiendo rangos (ej. Edad entre 18 y 65).
  • Longitud de Texto: Limita el número de caracteres (ej. Código de producto de 5 dígitos).
  • Fecha/Hora: Asegura que la entrada sea un formato de fecha válido y dentro de un período específico.
  • Personalizada: Permite usar una fórmula para definir reglas complejas.

Mensajes de Validación

Son cruciales para guiar al usuario y notificarle cuando comete un error.

Mensaje de Entrada (Input)

Aparece cuando la celda es seleccionada. Sirve como una instrucción preventiva.

2. Herramienta de Filtrado de Datos

El Filtrado es el proceso de mostrar solo las filas que cumplen con un criterio específico, mientras que el resto de las filas se ocultan temporalmente. Esto facilita el análisis y la visualización de subconjuntos de datos.

Tipos de Filtros y Aplicaciones

  • Filtro Básico (Por Valor): Muestra u oculta filas al seleccionar o deseleccionar elementos de una lista.
  • Filtro de Número: Aplica criterios lógicos a datos numéricos (ej. "Mayor que 100", "Entre 50 y 200").
  • Filtro de Texto: Aplica criterios a cadenas de texto (ej. "Comienza por 'Lic'", "Contiene la palabra 'ventas'").
  • Filtro de Color: Útil si la hoja de cálculo usa formato condicional; filtra por color de relleno o fuente.
  • Filtro Avanzado: Permite definir criterios complejos en un rango separado (usando AND/OR) para una extracción sofisticada.

Proceso: Aplicación de Validación

Pasos Generales:

  1. Seleccionar el rango de celdas a restringir (ej. la columna de Edades).
  2. Ir a la pestaña Datos (o similar, según el software).
  3. Hacer clic en la opción Validación de Datos.
  4. Elegir el Criterio (ej. Número Entero).
  5. Definir los Límites (ej. Mínimo 18, Máximo 65).
  6. Configurar los Mensajes de Entrada y Error.
