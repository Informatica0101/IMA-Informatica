¿Qué es la Validación de Datos?

Es una herramienta de Excel que permite controlar qué valores pueden ingresar los usuarios en una celda. Sirve para evitar errores humanos y mantener la integridad de la información.

Importancia

  • Evita datos incoherentes (ej. edades negativas).
  • Estandariza entradas (ej. nombres de ciudades).
  • Guía al usuario con mensajes de entrada.

Tipos de Validación: Listas

Muestra un menú desplegable con opciones predefinidas. Es el tipo más común para campos como "Género" o "Carrera".

Tipos de Validación: Números

Permite restringir la entrada a Números Enteros o Decimales dentro de un rango específico.

0 < x < 100

Ejemplo: Notas académicas

Longitud de Texto

Controla cuántos caracteres puede tener una celda. Útil para números de Identidad o Teléfonos fijos.

Ejemplo: La identidad debe tener exactamente 13 dígitos.

Mensaje de Entrada

Aparece cuando el usuario selecciona la celda, dándole instrucciones antes de que escriba nada.

Instrucción: Ingrese solo números del 1 al 10.

Mensaje de Error

Es la alerta que salta cuando el usuario intenta ingresar un dato que NO cumple las reglas. Existen 3 estilos:

  • Detener: Bloquea la entrada totalmente.
  • Advertencia: Avisa pero permite seguir.
  • Información: Solo comunica el error.

Ubicación en el Menú

Pestaña Datos → Herramientas de Datos → Validación de Datos

Círculos de Validación

Excel puede marcar con un círculo rojo los datos que ya estaban en la hoja y que NO cumplen con la nueva regla aplicada.

!

Listas Dependientes

Es una técnica avanzada donde las opciones de una lista cambian según lo que se seleccionó en otra lista anterior.

Limpieza

Para quitar todas las restricciones de una celda, debes entrar al menú de Validación y presionar el botón "Borrar todos".

Integridad

"La validación de datos es el primer escudo contra la información basura en una hoja de cálculo."
