¿Qué es la CLI?

CLI (Command Line Interface) es un método para interactuar con la computadora escribiendo comandos de texto, en lugar de usar iconos y el mouse (GUI).

¿Por qué usarla?

  • Velocidad: Ejecuta tareas complejas en segundos.
  • Potencia: Acceso a funciones que no están en los menús.
  • Automatización: Creación de scripts para tareas repetitivas.

El Prompt

Es el indicador que nos dice que la terminal está lista para recibir un comando.

C:\Users\Admin> _

Comando: DIR / LS

Muestra una lista de todos los archivos y carpetas en la ubicación actual.

dir (Windows) / ls (Linux/Mac)

Comando: CD

Change Directory: Sirve para entrar o salir de las carpetas.

cd Documentos (Entrar)
cd .. (Regresar una carpeta)

Comando: MKDIR / MD

Crea una carpeta nueva instantáneamente.

mkdir Tarea_Nueva

Comando: CLS / CLEAR

Limpia toda la pantalla de la terminal cuando está muy llena de texto.

Comandos de Red: IPCONFIG

Muestra tu dirección IP y otros datos de tu conexión a Internet.

ipconfig

Comandos de Red: PING

Verifica si tienes conexión con otro servidor (ej. Google).

ping www.google.com

Estructura de un Comando

comando [opciones] [argumento]

Ejemplo: ping -t 8.8.8.8

Atajos de Teclado

  • Flecha Arriba: Repetir comando anterior.
  • TAB: Autocompletar nombres de archivos.
  • CTRL + C: Detener un comando en ejecución.

Peligros de la Terminal

La terminal no suele pedir confirmación ("¿Está seguro?"). Un comando mal escrito puede borrar todo el disco duro en segundos. ¡Usar con precaución!
