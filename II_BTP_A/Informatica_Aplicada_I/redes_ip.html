¿Qué es una Red?

Es un conjunto de dispositivos interconectados entre sí que comparten recursos (archivos, impresoras), servicios (internet, correo) e información.

Tipos de Red (Alcance)

LAN

Red de Área Local (Hogar, Escuela).

MAN

Red de Área Metropolitana (Ciudad).

WAN

Red de Área Amplia (Internet).

La Dirección IP

Es la identificación única de un dispositivo en una red. Funciona como el número de teléfono o la dirección de una casa para que los datos sepan a dónde ir.

IPv4 vs IPv6

IPv4

4 números (0-255) separados por puntos.
Ej: 192.168.1.1

IPv6

8 grupos hexadecimales. Creado porque las IPv4 se agotaron.
Ej: 2001:0db8:...

Dispositivos de Red

  • Router: Conecta diferentes redes y dirige el tráfico.
  • Switch: Conecta múltiples dispositivos en una misma red local.
  • Punto de Acceso (AP): Proporciona conexión inalámbrica (Wi-Fi).

Ejercicio Práctico 1

Averigua la dirección IP local de tu dispositivo. (En Windows usa el comando ipconfig en la terminal).

Topologías de Red

Es la forma en que están conectados físicamente los cables:

Estrella (Más común)
Bus
Anillo
Malla

Protocolos

Son las reglas que permiten la comunicación:

  • HTTP/HTTPS: Páginas web.
  • FTP: Transferencia de archivos.
  • TCP/IP: Protocolo base de Internet.

Seguridad en Redes

Uso de contraseñas fuertes (WPA2/WPA3) y Firewalls para evitar accesos no autorizados a nuestra información personal.

El Modelo OSI

Divide la comunicación en 7 capas para facilitar su estudio y estandarización, desde el cable físico hasta la aplicación del usuario.

Un Mundo Conectado

"Las redes son las arterias de la sociedad moderna. Sin ellas, no existiría la nube ni el trabajo global."
