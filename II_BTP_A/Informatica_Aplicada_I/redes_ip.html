¿Qué es una Red de Computadoras?

Una red de computadoras es un sistema que interconecta dos o más dispositivos para permitirles compartir recursos e intercambiar información.

  • Permite la comunicación y el intercambio de datos.
  • Formada por hardware (equipos) y software (programas).

Tipos de Redes (por Alcance)

PAN (Red de Área Personal)

Es la más pequeña, diseñada para un solo usuario y sus dispositivos cercanos.

LAN (Red de Área Local)

Conecta dispositivos dentro de un área limitada, como una casa, una oficina o un edificio escolar.

MAN (Red de Área Metropolitana)

Es más grande que una LAN, cubriendo una ciudad o un campus universitario.

WAN (Red de Área Amplia)

Conecta redes a través de grandes distancias geográficas, como regiones o países.

Topologías de Red

La forma en que los dispositivos están conectados en una red.

Estrella

Cada dispositivo se conecta a un nodo central (switch).

Bus

Todos comparten un cable principal.

Anillo

Los dispositivos se conectan en un círculo cerrado.

Es la forma en que está diseñada la red, sea en el plano físico o lógico. El concepto de red puede definirse como ¨conjunto de nodos interconectados¨.

Protocolos de Red

Los protocolos de red son conjuntos de reglas y estándares que los dispositivos deben seguir para comunicarse de manera efectiva.

TCP/IP

El protocolo más importante y la base de Internet.

Direccionamiento IP

Una dirección IP es la identificación única de cada dispositivo en una red.

IP Dinámica (DHCP)

Asignada automáticamente por el router.

IP Estática

Asignada manualmente. Útil para servidores.

Componentes de Configuración IP

Dirección IP

Identificación única del dispositivo.

Máscara de Subred

Define la porción de red y de host.

Puerta de Enlace (Gateway)

Dirección del router para salir de la red local.

Servidores DNS

Traducen nombres de dominio a IPs.

Curiosidad Digital: El Ping

Una herramienta para verificar la conectividad de red.

El comando `ping` envía pequeños paquetes de datos a un destino y mide el tiempo que tardan en regresar, indicando si hay conexión y su velocidad.

Ejemplo en terminal: `ping google.com`
