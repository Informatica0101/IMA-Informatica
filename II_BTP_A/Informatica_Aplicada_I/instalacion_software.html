¿Qué es la Instalación?

Es el proceso de preparar un programa para ser ejecutado en el sistema. Implica copiar archivos, registrar librerías y configurar parámetros iniciales.

Tipos de Instalación

  • Típica: Configuración estándar básica.
  • Personalizada: El usuario elige componentes.
  • Silenciosa: Sin intervención del usuario.

¿Qué es la Virtualización?

Es la tecnología que permite crear una representación virtual de recursos físicos (como una computadora completa) dentro de otra computadora real.

La Máquina Virtual (VM)

Es un software que emula una computadora física. Tiene su propia CPU virtual, memoria, disco y puede ejecutar un Sistema Operativo independiente.

El Hypervisor

Es el software "maestro" que gestiona las máquinas virtuales y reparte los recursos del hardware físico entre ellas.

Tipos: Nativo (Tipo 1) y Alojado (Tipo 2).

Hypervisor Tipo 2

Se instala como un programa más dentro de Windows o Linux. Es ideal para estudiantes y laboratorios.


VirtualBox

VMware

Ventajas de Virtualizar

  • Aislamiento: Si la VM falla, el PC real no se afecta.
  • Snapshots: Guardar el estado actual y volver atrás.
  • Consolidación: Muchos SO en un solo servidor físico.

Requisitos de Hardware

Virtualizar consume muchos recursos físicos. Se recomienda:

Mucha RAM (8GB+)
CPU con Virtualización
SSD rápido
Espacio en disco

Uso en la Nube

La mayoría de los servidores en Internet son realmente Máquinas Virtuales (vía AWS, Azure, Google Cloud).

Instalación en la VM

Para instalar un SO en una VM se suele usar un archivo .ISO, que es la copia digital de un disco de instalación.

Host vs Guest

  • Anfitrión (Host): Tu computadora real.
  • Invitado (Guest): El sistema dentro de la VM.

Importancia

"La virtualización permite probar software peligroso o nuevos sistemas sin poner en riesgo nuestra herramienta de trabajo real."
