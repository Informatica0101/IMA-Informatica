El Corazón del Software

El Sistema Operativo (SO) es el programa base que gestiona todos los recursos de hardware y software de la computadora.

Sin un SO, el hardware es inútil para el usuario promedio.

Funciones Vitales

Gestión de Procesos

Asigna tiempo de CPU a cada programa.

Gestión de Memoria

Administra la RAM disponible.

Gestión de Archivos

Crea y organiza la estructura en discos.

Interfaz de Usuario

Permite al humano interactuar con la máquina.

El Kernel (Núcleo)

Es la parte más interna del SO. Gestiona el acceso seguro al hardware. Siempre reside en la memoria RAM mientras la PC está encendida.

Tipos de SO

Escritorio: Windows, macOS, Linux.
Móviles: Android, iOS.
Servidores: Linux Server, Windows Server.

Multitarea y Multiusuario

Multitarea: Capacidad de correr varios programas al mismo tiempo.

Multiusuario: Permite que varias personas usen el sistema simultáneamente.

Ejercicio Práctico 1

Abre el "Administrador de Tareas" en tu computadora y observa cuántos procesos se están ejecutando y cuánta RAM consumen.

Controladores (Drivers)

Son pequeños programas que enseñan al SO cómo comunicarse con un hardware específico (impresora, tarjeta de video, etc.).

Sistemas de Archivos

Es la forma en que el SO organiza los datos en el disco.
Ejemplos: NTFS (Windows), APFS (macOS), EXT4 (Linux).

Seguridad del SO

El SO debe protegerse contra malware, gestionar permisos de usuario y cifrar datos sensibles.

Virtualización

Permite ejecutar un SO dentro de otro como si fuera una aplicación (Máquina Virtual). Útil para pruebas y servidores.

Evolución Constante

"Los sistemas operativos de hoy son el resultado de décadas de optimización para hacer la tecnología invisible y potente a la vez."
