¿Qué es el Pseudocódigo?

Es una herramienta de programación que permite escribir la lógica de un algoritmo usando una mezcla de lenguaje natural y palabras clave similares a un lenguaje de programación.

  • 📝 "Pseudo" significa falso, por lo que es un "falso código".
  • 💻 No lo ejecuta la computadora directamente.
  • 🧠 Está diseñado para ser leído y entendido por humanos.

Características

INDEPENDIENTE

Se puede traducir a cualquier lenguaje de programación (C++, Python, Java).

ESTRUCTURADO

Sigue una secuencia lógica con un inicio y un fin bien definidos.

FÁCIL MANTENIMIENTO

Es más sencillo corregir la lógica en texto que en un diagrama complejo.

PALABRAS CLAVE

Usa términos como: Leer, Escribir, Si-Entonces, Mientras, Para.

Simbología y Palabras Clave

Inicio / Fin Marcan el comienzo y el término del algoritmo.
Leer Captura datos ingresados por el usuario (Entrada).
Escribir Muestra información en pantalla (Salida).
<- Símbolo de asignación (Ej: x <- 10).

Ejemplo 1: Saludo

Algoritmo que pide un nombre y saluda.

Algoritmo Saludo
  Escribir "Ingresa tu nombre:"
  Leer nombre
  Escribir "¡Hola ", nombre, "!"
FinAlgoritmo

Ejemplo 2: Suma Básica

Algoritmo Sumar
  Escribir "Ingrese primer número:"
  Leer n1
  Escribir "Ingrese segundo número:"
  Leer n2
  resultado <- n1 + n2
  Escribir "La suma es: ", resultado
FinAlgoritmo

Ordenación Simbólica

Se refiere a la organización jerárquica y secuencial de los símbolos en el diseño de algoritmos.

1 Secuencia: Pasos uno tras otro.
2 Selección: Caminos alternativos (Si-Entonces).
3 Iteración: Repetición de pasos (Bucles).

Decisiones en Pseudocódigo

Se usa para ejecutar instrucciones solo si se cumple una condición.

Si edad >= 18 Entonces
  Escribir "Eres mayor de edad"
Sino
  Escribir "Eres menor de edad"
FinSi

¿Cuál elegir?

Pseudocódigo

Ideal para algoritmos largos y complejos con muchos cálculos.

Diagrama de Flujo

Ideal para visualizar el flujo lógico y las decisiones rápidamente.

"Un buen programador domina ambos."

Ejemplo Final: Área de Triángulo

Algoritmo AreaTriangulo
  Definir base, altura, area Como Real
  Escribir "Ingrese la base:"
  Leer base
  Escribir "Ingrese la altura:"
  Leer altura
  area <- (base * altura) / 2
  Escribir "El área es: ", area
FinAlgoritmo
