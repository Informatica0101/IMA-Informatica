¿Qué es Programar?

Es el proceso de crear un conjunto de instrucciones precisas que una computadora puede seguir para realizar una tarea o resolver un problema.

Es como darle una receta de cocina a una máquina muy rápida pero poco intuitiva.

¿Qué es un Algoritmo?

Un algoritmo es una secuencia de pasos lógicos, finitos y ordenados para resolver un problema.

Ejemplo diario: Cepillarse los dientes, cambiar una llanta, preparar café.

Características

Finito

Debe tener un fin, un número determinado de pasos.

Preciso

Cada paso debe ser claro y sin ambigüedad.

Definido

Mismas entradas deben producir mismos resultados.

General

Debe servir para todos los casos del problema.

Tipos de Algoritmos

Cualitativos

Pasos descritos con palabras (ej. receta).

Cuantitativos

Involucran cálculos numéricos (ej. área de un círculo).

Lenguajes de Programación

Son los "idiomas" que usamos para hablar con la computadora.

Estructura Secuencial

Es la forma más simple de programación: los pasos se ejecutan uno tras otro, de arriba hacia abajo.

Variables y Constantes

Variables

Espacios de memoria que pueden cambiar de valor (ej. puntaje, edad).

Constantes

Valores que no cambian durante el programa (ej. PI = 3.1416).

Tipos de Datos Simples

  • Entero: Números sin decimales (1, 2, -5).
  • Real: Números con decimales (3.14, 0.5).
  • Carácter: Una sola letra o símbolo ('A', '$').
  • Cadena: Texto ("Hola Mundo").
  • Lógico: Verdadero o Falso.
