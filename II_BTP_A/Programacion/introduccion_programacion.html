Introducción a la Lógica

La lógica es la base del pensamiento computacional. Es la capacidad de razonar de manera estructurada para resolver problemas complejos de forma coherente.

¿Por qué es importante la Lógica?

Resolución de Problemas

Permite descomponer un problema grande en partes pequeñas y manejables.

Eficiencia

Ayuda a encontrar la ruta más corta y efectiva para llegar a una solución.

Estructura

Proporciona el orden necesario para que las instrucciones sean claras.

Base Tecnológica

Es el lenguaje fundamental que permite a los humanos comunicarse con las máquinas.

¿Qué es la Programación?

Programar es el proceso de diseñar, codificar, depurar y mantener el código fuente de programas computacionales. Básicamente, es dar instrucciones precisas a una computadora.

Lenguajes de Programación

Son herramientas que permiten traducir nuestras ideas al lenguaje de las máquinas.

  • Bajo Nivel: Directamente relacionado con el hardware (Ej: Lenguaje Ensamblador).
  • Alto Nivel: Más cercano al lenguaje humano, fácil de aprender (Ej: Python, C++, Java).
  • Propósito: Pueden ser para web, aplicaciones móviles, ciencia de datos o sistemas.

¿Qué es un Algoritmo?

Es una secuencia lógica, finita y definida de pasos para resolver un problema o realizar una tarea específica.

"Un algoritmo es como una receta de cocina: sigues los pasos en orden para obtener un resultado delicioso".

Características de un Algoritmo

1. Preciso: Debe indicar el orden exacto de realización de cada paso.
2. Definido: Si se sigue dos veces con los mismos datos, se obtiene el mismo resultado.
3. Finito: Debe tener un número determinado de pasos; debe terminar en algún momento.

Representación de Algoritmos

Lenguaje Natural

Descripción simple paso a paso usando nuestro propio idioma.

Pseudocódigo

Uso de palabras clave que simulan código real pero en español/inglés simple.

Diagramas de Flujo

Representación gráfica mediante símbolos estandarizados.

Estructuras Lógicas Básicas

Todo programa o algoritmo se construye usando tres estructuras fundamentales:

  • Secuencial: Las instrucciones se ejecutan una tras otra en el orden en que aparecen.
  • Decisión (Si/Entonces): Permite elegir entre dos o más caminos según se cumpla una condición.
  • Repetición (Bucles): Permite ejecutar un bloque de instrucciones varias veces.

Conclusión

Dominar la lógica y entender los algoritmos es el primer paso para convertirte en un desarrollador exitoso. La programación no se trata solo de escribir código, sino de resolver problemas de manera inteligente.
