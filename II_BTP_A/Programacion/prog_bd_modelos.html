¿Qué es una Base de Datos?

Es un banco de datos o conjunto de datos que pertenecen al mismo contexto, almacenados sistemáticamente para su posterior uso.

Son programas que administran información, la ordenan y la hacen fácil de buscar y encontrar.

Usos y Funciones

  • Almacenamiento: Guardar grandes volúmenes de datos.
  • Recuperación: Acceder a la información de forma rápida y flexible.
  • Organización: Reordenar datos según la necesidad.
  • Distribución: Imprimir o compartir información en diversos formatos.

Enfoque Declarativo

Los lenguajes orientados a bases de datos son declarativos y no imperativos.

El problema es "qué" se quiere buscar o hacer en la base de datos, y no se enfatiza el "cómo" realizar el proceso paso a paso.

Balance de Uso

Ventajas:

  • Orden y estructura de datos.
  • Búsquedas instantáneas.
  • Comunicación eficiente.

Sistemas Gestores (SGBD)

Permiten almacenar y posteriormente acceder a los datos de forma rápida y estructurada mediante software especializado.

Ejemplo: SQL Server, MySQL o Access sirven como intermediarios entre el usuario y los datos físicos.

Ejercicio 1: Aplicaciones

Investiga y anota en tu cuaderno 3 ejemplos de bases de datos que utilices en tu vida diaria (ej: redes sociales, bancos, etc.) y qué datos crees que almacenan.

Bases de Datos por su Variabilidad

Estáticas

De sólo lectura. Usadas para datos históricos y estudio de comportamiento en el tiempo.

Según su Contenido

  • Bibliográficas: Contienen un "representante" (autor, título) pero no el texto completo.
  • De texto completo: Almacenan las fuentes primarias íntegras.
  • Directorios: Guías telefónicas o empresariales en formato electrónico.
  • Bancos de imágenes: Colecciones de fotos, audio o multimedia.
