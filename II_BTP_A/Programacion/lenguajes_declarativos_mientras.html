Lenguajes Declarativos

A diferencia de los imperativos (que dicen CÓMO hacer algo), los declarativos se centran en QUÉ se quiere obtener.

Principal ejemplo: SQL (Lenguaje de Consulta Estructurado).

Diferencia de Enfoque

Imperativo

"Ve a la cocina, abre la nevera, saca el agua, tráela".

Declarativo

"Quiero un vaso de agua". (El sistema decide los pasos).

El Bucle MIENTRAS (WHILE)

Es una estructura de control que repite un bloque de código mientras una condición sea VERDADERA.

Se usa cuando NO sabemos exactamente cuántas veces se repetirá el ciclo.

Estructura del WHILE

MIENTRAS condicion == VERDADERA HACER

  // Instrucciones

  // Importante: Cambiar la condición dentro

FINMIENTRAS

Condición de Salida

Para evitar un bucle infinito, dentro del MIENTRAS debe ocurrir algo que eventualmente haga que la condición sea falsa.

Ejemplo: Incrementar un contador o leer un nuevo valor.

Ejercicio Práctico 1

Escribe el pseudocódigo para un algoritmo que pida una contraseña y se repita MIENTRAS la contraseña sea incorrecta.

Diferencia FOR vs WHILE

PARA (FOR)

Número de vueltas fijo y conocido.

MIENTRAS (WHILE)

Depende de un evento o condición externa.

Validación de Datos

El bucle WHILE es ideal para validar que el usuario ingrese datos correctos:

LEER nota

MIENTRAS nota < 0 O nota > 100 HACER

  ESCRIBIR "Error, ingrese nota válida"

  LEER nota

FINMIENTRAS

Sentinelas

Un sentinela es un valor especial que se usa para terminar un bucle (ej. "Ingresa 0 para salir").

Ejemplo en SQL

SQL es declarativo. No decimos cómo buscar, solo qué queremos:

SELECT nombre, nota

FROM alumnos

WHERE nota >= 70;

Poder de Adaptación

"Los bucles condicionales permiten que los programas reaccionen al entorno y a la interacción impredecible del usuario."

Ejercicio Avanzado

Cajero Automático (Simulación):

saldo <- 1000

MIENTRAS saldo > 0 HACER

  LEER retiro

  IF retiro <= saldo THEN

    saldo <- saldo - retiro

  ELSE

    PRINT "Fondos insuficientes"

  ENDIF

FINMIENTRAS
