Paso 1: Análisis del Problema

Esta fase requiere una clara definición, donde se contemple exactamente lo que debe hacer el programa y el resultado o solución deseada.

❓ Preguntas clave:

  • 📊 ¿Qué cantidad y tipo de datos de entrada se requieren?
  • 📉 ¿Qué cantidad y tipo de salidas se desean?
  • 🧮 ¿Qué métodos y fórmulas se necesitan para procesar los datos?

Ejemplo de Análisis: Área del Círculo

E Entrada: Radio del círculo (Número Real).

Algoritmos en Lenguaje Natural

Es la forma de describir los pasos de un algoritmo utilizando nuestro idioma cotidiano (español), sin reglas rígidas de programación.

✅ Ventajas:

Permite expresar la lógica de manera clara y comprensible para cualquier persona antes de empezar a escribir código.

Ejemplo: Hacer una Llamada

1. Inicio.

2. Desbloquear el teléfono.

3. Abrir la aplicación de contactos.

4. Buscar el nombre de la persona.

5. Presionar el botón de llamar.

6. Esperar a que contesten.

7. Hablar y luego colgar.

8. Fin.

Ejemplo: Calcular Promedio

1. Pedir la nota del primer parcial.

2. Pedir la nota del segundo parcial.

3. Sumar ambas notas.

4. Dividir el resultado entre 2.

5. Mostrar el promedio final.

Características Clave

No es ambiguo

Cada paso debe ser claro y tener una sola interpretación.
