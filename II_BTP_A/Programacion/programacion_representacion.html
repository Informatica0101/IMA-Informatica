Programación del Algoritmo

Una vez que el algoritmo está diseñado, debe ser transformado en un programa ejecutable por la computadora.

1. Codificación

1. Codificación

Consiste en escribir el algoritmo utilizando un lenguaje de programación específico (como C++, Python o JavaScript).

  • Se siguen las reglas gramaticales (sintaxis) del lenguaje.
  • Se definen variables y estructuras de control.
  • El resultado es un programa fuente.

3. Comprobación (Prueba)

Consiste en verificar que el programa funciona correctamente y produce los resultados deseados.

✅ Detectar errores de sintaxis o lógica.

Técnicas de Representación

Existen diferentes formas de documentar la lógica de un algoritmo:

🗣️ Lenguaje Natural

Repaso: Lenguaje Natural

Es la descripción paso a paso del proceso usando nuestro idioma cotidiano.

"1. Preguntar el nombre.
2. Leer el nombre.
3. Saludar diciendo Hola y el nombre."

Pseudocódigo

Es un lenguaje intermedio entre el lenguaje humano y el código de programación.

  • Usa palabras clave (Leer, Escribir, Si, Entonces).
  • Es independiente del lenguaje de programación.
  • Facilita la conversión posterior a código real.

Diagramas de Flujo

Es la representación gráfica de un algoritmo. Utiliza símbolos estándar conectados por flechas.

🔘 Óvalo: Inicio/Fin

La importancia del "Paso a Paso"

Realizar la conversión de algoritmos a instrucciones precisas es la clave para que la computadora no cometa errores.

"Si el algoritmo tiene un error de lógica, el programa también lo tendrá."
