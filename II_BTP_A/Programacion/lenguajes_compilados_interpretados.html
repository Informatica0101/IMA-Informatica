El Proceso de Compilación

Un compilador es un programa que traduce un programa escrito en un lenguaje de alto nivel a lenguaje de máquina.

Traducción Previa: El compilador traduce el código completo antes de que el programa se ejecute, creando un archivo llamado "código objeto".

Ventajas de los Lenguajes Compilados

  • Velocidad: El programa se ejecuta mucho más rápido porque ya está en lenguaje de máquina.
  • Optimización: El compilador puede mejorar el código durante la traducción.
  • Seguridad: El código fuente no necesita ser distribuido con el ejecutable.

El Proceso de Interpretación

Un intérprete traduce y ejecuta el programa línea por línea, conforme sea necesario durante el proceso de datos.

Traducción en Tiempo Real: No se graba un código objeto; el intérprete debe estar presente cada vez que se corre el programa.

Ventajas de los Lenguajes Interpretados

  • Tiempo de Diseño: Es más rápido desarrollar porque no hay que esperar a que el código se compile tras cada cambio.
  • Depuración: Es más fácil encontrar y corregir errores sobre la marcha.
  • Multiplataforma: El mismo código fuente puede correr en distintos sistemas que tengan el intérprete.

Compilación vs Interpretación

Compilados

Ejecución rápida, requieren paso de traducción previo. Ejemplos: C++, Java (Híbrido).

Interpretados

Ejecución más lenta, desarrollo más ágil. Ejemplos: Python, PHP, JavaScript.

Estructuras de Control: Bucles

Un bucle es una estructura que permite repetir un bloque de instrucciones.

¿Cuándo se utilizan? Se usan cuando necesitamos realizar una misma tarea múltiples veces sobre diferentes datos o simplemente repetir un proceso un número determinado de veces.

El Bucle "Para"

Se utiliza específicamente cuando conocemos de antemano el número exacto de veces que queremos que se repita la acción.

Para variable = inicio Hasta fin Con Paso incremento Hacer
  Instrucciones...
FinPara

Componentes del Bucle Para

  • Variable de control: Almacena el número de la repetición actual.
  • Valor Inicial y Final: Definen el rango de ejecución.
  • Incremento: Define cuánto aumenta la variable en cada vuelta.

Ejemplo: Imprimir números del 1 al 5

Para i = 1 Hasta 5 Hacer
  Escribir i
FinPara

Resultado:

1, 2, 3, 4, 5

Sintaxis en C, C++ y JavaScript

Curiosamente, estos tres lenguajes comparten la misma sintaxis para el bucle for:

for (int i = 1; i <= 5; i++) {
  // Instrucciones en C/C++
}

for (let i = 1; i <= 5; i++) {
  // Instrucciones en JavaScript
}
