Lenguajes Compilados

Un programa traductor (compilador) procesa TODO el código fuente y genera un archivo ejecutable antes de que el programa corra.

Ventaja: Son mucho más rápidos al ejecutarse.

Ejemplos: C, C++, Pascal.

Lenguajes Interpretados

Un intérprete lee y ejecuta el código línea por línea en tiempo real. No generan un ejecutable previo.

Ventaja: Fáciles de depurar y multiplataforma.

Ejemplos: Python, JavaScript, Ruby.

¿Qué es un Bucle?

Un bucle o ciclo es una estructura de control que permite repetir un bloque de código varias veces mientras se cumpla una condición.

El Bucle PARA (FOR)

Se utiliza cuando sabemos exactamente cuántas veces queremos que se repita la acción.

Ejemplo: Escribir los números del 1 al 10.

Estructura del Bucle FOR

PARA i <- 1 HASTA 10 CON PASO 1 HACER

  ESCRIBIR i

FINPARA

Contadores

Es una variable que aumenta su valor en una cantidad constante (generalmente de 1 en 1).

c = c + 1

Acumuladores

Variable que almacena sumas variables de valores. Se usa para obtener totales.

suma = suma + precio_articulo

Diferencia Visual

Compilado

Código -> Ejecutable -> Resultado final rápido.

Interpretado

Código -> Lectura línea a línea -> Resultado.

Bucles Infinitos

Ocurren cuando la condición de salida NUNCA se cumple. El programa se "congela" o consume toda la memoria.

Eficiencia

"Un buen programador elige el bucle adecuado para cada tarea, evitando repeticiones innecesarias."

Ejemplo: Acumulador

suma <- 0

PARA i <- 1 HASTA 3 HACER

  LEER nota

  suma <- suma + nota

FINPARA

ESCRIBIR "El total es: ", suma
