¿Qué es un Lenguaje de Programación?

Es un sistema de símbolos y reglas (sintaxis) que permiten a un programador dar instrucciones a una computadora para que procese datos y resuelva problemas.

Niveles de Lenguaje

Bajo Nivel

Cercanos al hardware (Binario, Ensamblador). Difíciles para humanos.

Alto Nivel

Cercanos al lenguaje humano (C++, Python, Java). Fáciles de leer.

Sintaxis y Semántica

Sintaxis: Reglas de escritura (puntos, comas, paréntesis).

Semántica: El significado o lógica detrás de lo que se escribe.

Tipos por Ejecución

Compilados

Se traducen todos a la vez antes de ejecutarse (C++).

Interpretados

Se traducen línea por línea mientras corren (Python).

Paradigmas

Es el estilo o enfoque que se usa para programar:

  • Imperativo: Pasos detallados.
  • Declarativo: Basado en el resultado deseado.
  • Orientado a Objetos: Basado en entidades del mundo real.

IDE (Entorno de Desarrollo)

Es el software que usamos los programadores para escribir y probar código. Incluye editor de texto, compilador y depurador.

Ejemplos: Visual Studio Code, Dev-C++, PyCharm.

Código Fuente

Es el archivo de texto que contiene las instrucciones escritas por el programador antes de ser traducidas a lenguaje de máquina.

Errores de Sintaxis

Ocurren cuando no respetamos las reglas de escritura (faltó un punto y coma, paréntesis sin cerrar). El programa no podrá ejecutarse.

El Futuro

"Los lenguajes están evolucionando para ser más naturales, integrando Inteligencia Artificial."
