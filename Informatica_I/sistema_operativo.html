¿Qué es un Sistema Operativo?

Software esencial que gestiona hardware y software, permitiendo la ejecución de programas.

  • Intermediario entre hardware y usuario/aplicaciones.
  • Crea un entorno de trabajo eficiente.

Funciones Principales

El SO es el cerebro que coordina todo.

Gestión de Procesos

  • Administra la ejecución de programas.
  • Asigna tiempo de CPU (planificación).
  • Permite la multitarea.

Gestión de Memoria

  • Asigna y libera RAM a programas.
  • Controla el uso de la memoria.
  • Optimiza el rendimiento.

Funciones Principales (Cont.)

Organización de datos y comunicación con periféricos.

Gestión de Archivos

  • Organiza datos en discos (archivos/directorios).
  • Permite crear, leer, escribir, eliminar.
  • Gestiona permisos de acceso.

Gestión de Dispositivos (I/O)

  • Controla comunicación con periféricos.
  • Interfaz uniforme para programas.

Funciones Principales (Cont.)

Cómo interactuamos y cómo nos protege el sistema.

Interfaz de Usuario

  • Permite interacción (CLI o GUI).
  • CLI: comandos de texto.
  • GUI: ventanas, iconos, menús.

Redes y Seguridad

  • Comunicación entre computadoras.
  • Protege contra accesos no autorizados.
  • Gestiona permisos y cifrado.

Tipos de Sistemas Operativos

Clasificaciones según su capacidad y propósito.

Por Usuarios

  • Monousuario: Un usuario a la vez (ej. MS-DOS).
  • Multiusuario: Varios usuarios simultáneamente (ej. Linux, Windows Server).

Por Tareas

  • Monotarea: Una tarea a la vez (ej. MS-DOS).
  • Multitarea: Múltiples programas concurrentes (ej. Windows, macOS).

Tipos de Sistemas Operativos (Cont.)

Cómo están construidos y para qué se usan.

Por Estructura

  • Centralizados, de Red, Distribuidos.

Por Propósito

  • Escritorio, Servidor, Móviles, Embebidos, Tiempo Real.

Componentes Clave

Las partes esenciales que hacen funcionar un SO.

Kernel (Núcleo)

  • Corazón del SO.
  • Gestión de CPU, memoria, E/S.
  • Reside permanentemente en memoria.

Shell / Interfaz de Usuario

  • Permite interacción con el kernel.
  • CLI (comandos) o GUI (visual).

Componentes Clave (Cont.)

Más elementos fundamentales del SO.

  • Sistema de Archivos: Organiza datos en discos.
  • Gestor de Procesos: Controla programas en ejecución.
  • Gestor de Memoria: Asigna y libera RAM.
  • Gestor de Dispositivos: Comunicación con hardware (drivers).

Ejemplos de SO Populares

Exploraremos los sistemas operativos más utilizados.

  • Microsoft Windows
  • macOS
  • Linux
  • Android
  • iOS

Microsoft Windows

El sistema operativo de escritorio más extendido globalmente.

  • Interfaz gráfica de usuario (GUI) amigable.
  • Amplia compatibilidad con software y hardware.
  • Versiones populares: Windows 10, Windows 11.

macOS

El sistema operativo de Apple para sus computadoras Mac.

  • Diseño elegante e intuitivo.
  • Alta estabilidad y seguridad.
  • Integración profunda con el ecosistema de Apple.

Linux

Un sistema operativo de código abierto y gratuito.

  • Gran flexibilidad y personalización.
  • Popular en servidores, supercomputadoras y desarrollo.
  • Múltiples "distribuciones" (ej. Ubuntu, Fedora, Debian).

Android

El sistema operativo móvil más utilizado, desarrollado por Google.

  • Basado en el kernel de Linux.
  • Amplia variedad de dispositivos y fabricantes.
  • Gran ecosistema de aplicaciones (Google Play Store).

iOS

El sistema operativo móvil de Apple para iPhone y iPad.

  • Interfaz de usuario simple y fluida.
  • Foco en la seguridad y la privacidad.
  • Optimizado para el hardware de Apple.

¿Sabías que...?

  • El primer SO de uso general fue de General Motors en 1956.
  • "Linux" viene de Linus Torvalds y "Unix".
  • Android tiene más del 70% de cuota de mercado móvil.
  • Los SO modernos usan "paginación" y "memoria virtual".
  • Los "BSOD" (pantalla azul de la muerte) en Windows son de seguridad.
  • Chrome OS de Google está basado en Linux y usa el navegador Chrome como su interfaz principal.
  • Los sistemas operativos embebidos son cruciales en IoT (Internet de las Cosas).
