¿Qué es un S.O.?

Es el software principal que gestiona los recursos de hardware y provee servicios a los programas de aplicación. Es el intermediario entre el usuario y la computadora.

Funciones Principales

  • Gestión de la CPU (Procesos).
  • Gestión de la Memoria RAM.
  • Gestión de archivos y carpetas.
  • Interfaz de Usuario (GUI o CLI).

Componentes: El Kernel

El Núcleo o Kernel es el corazón del S.O. Se encarga de las tareas más críticas como la comunicación directa con el hardware y la seguridad.

Multitarea (Multitasking)

Capacidad del S.O. para ejecutar varios procesos "al mismo tiempo" repartiendo los ciclos de reloj de la CPU entre ellos.

Interfaz de Usuario (UI)

  • GUI: Gráfica (iconos, ventanas, mouse).
  • CLI: Comandos (texto, terminal).

Controladores (Drivers)

Pequeños programas que permiten al S.O. comunicarse con dispositivos de hardware específicos como impresoras o tarjetas de video.

Sistemas de Archivos

Es la forma en que el S.O. organiza los datos en el disco duro. Ejemplos: NTFS (Windows), APFS (Mac), EXT4 (Linux).

El Proceso de Arranque (Boot)

Desde que presionas el botón de encendido hasta que aparece el escritorio, el S.O. realiza una serie de pruebas de hardware (POST) y carga el núcleo en la RAM.
