¿Qué son los Periféricos?

Son dispositivos auxiliares e independientes conectados a la unidad central de procesamiento de una computadora.

Permiten la comunicación entre la computadora y el mundo exterior.

Clasificación Principal

Entrada

Captan y envían datos a la CPU.

Salida

Muestran o proyectan resultados.

Mixtos / Almacenamiento

Realizan ambas funciones o guardan datos.

Periféricos de Entrada

Teclado

Permite ingresar texto y comandos.

Mouse

Dispositivo apuntador para la interfaz.

Micrófono

Ingreso de señales de audio.

Cámara Web

Captura de imagen y video.

Periféricos de Salida

Monitor

Muestra la interfaz visual.

Impresora

Plasmado de datos en papel.

Parlantes

Salida de audio y sonidos.

Auriculares

Salida de audio individual.

Periféricos Mixtos

Son aquellos que pueden enviar datos a la CPU y también recibir resultados.

  • Pantallas táctiles.
  • Impresoras multifuncionales (con escáner).
  • Módems / Routers (comunicación bidireccional).
  • Memorias USB y Discos Externos.

Ejercicio Práctico

Clasifica los siguientes periféricos en tu cuaderno (Entrada o Salida):

  1. Escáner
  2. Proyector (Data Show)
  3. Joystick (Mando)
  4. Monitor Táctil
  5. Plotter

Puertos de Conexión

Los periféricos se conectan mediante diversos estándares:

USB

HDMI / DisplayPort

Bluetooth / Wi-Fi

El Escáner

Función: Entrada

Digitaliza documentos físicos convirtiéndolos en impulsos eléctricos que la computadora puede procesar como archivos de imagen o texto.

El Monitor

Función: Salida

Es el periférico de salida más importante. Utiliza una cuadrícula de píxeles para formar la interfaz visual que vemos.

Periféricos para Accesibilidad

Existen periféricos especializados para personas con discapacidad:

  • Teclados Braille.
  • Sistemas de seguimiento ocular.
  • Sintetizadores de voz.

Mantenimiento Básico

  • Limpiar polvo con aire comprimido.
  • No usar líquidos directamente.
  • Conectar y desconectar con cuidado.

Lógica de Transferencia

WHILE dispositivo_conectado == TRUE

  IF evento_usuario == TRUE THEN

    SEND_SIGNAL_TO_CPU(data)

    WAIT_FOR_RESPONSE()

  ENDIF

ENDWHILE
