¿Qué es la Informática?

Es la ciencia que estudia el tratamiento automático y racional de la información mediante el uso de sistemas computacionales.

Proviene de las palabras: INFORMACIÓN + AUTOMÁTICA.

La Computadora

Es una máquina electrónica capaz de:

  • Recibir datos (Entrada).
  • Procesar datos (Cálculos/Lógica).
  • Devolver resultados (Salida).
  • Almacenar información.

Componentes del Sistema

Hardware

Parte física (lo que se puede tocar).

Software

Parte lógica (programas e instrucciones).

Humanware

El factor humano (usuarios y especialistas).

Hardware

Corresponde a todas las partes físicas y tangibles de una computadora.

CPU (Cerebro)
Memoria RAM
Disco Duro (HDD/SSD)
Placa Madre

Software

Es el conjunto de programas y datos que permiten que el hardware funcione.

De Sistema: Windows, Android, Linux.
De Aplicación: Word, Chrome, WhatsApp.

Tipos de Computadoras

Escritorio (Desktop)

Diseñadas para permanecer fijas en un escritorio. Son potentes y modulares.

Portátiles (Laptops)

Compactas y con batería, permiten movilidad total para el usuario.

Tabletas y Smartphones

Estos dispositivos también son computadoras. Tienen procesador, memoria, almacenamiento y ejecutan software especializado.

Unidades de Almacenamiento

HDD (Disco Duro)

Usa discos magnéticos. Más capacidad, menos velocidad.

SSD (Estado Sólido)

Usa chips de memoria. Mucho más rápido y resistente.
