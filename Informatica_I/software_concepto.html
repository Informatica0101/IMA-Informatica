¿Qué es el Software?

Es el componente lógico del sistema. Comprende el conjunto de programas, datos y protocolos que permiten que el hardware realice tareas inteligentes.

Es la parte intangible e inmaterial de la computadora.

Clasificación del Software

El software se divide principalmente en tres categorías según su función:

SISTEMA
APLICACIÓN
PROGRAMACIÓN

Software de Sistema

Sirve de interfaz entre el usuario, las aplicaciones y el hardware. Gestiona los recursos básicos del equipo.

Ejemplos: Sistemas Operativos (Windows, Linux, Android), Controladores (Drivers) y herramientas de diagnóstico.

Software de Aplicación

Son los programas diseñados para que el usuario realice tareas específicas (productividad, ocio, comunicación).

Office
Navegadores
Videojuegos
Editores Multimedia

Software de Programación

Herramientas que permiten a los desarrolladores crear otros programas usando lenguajes específicos.

Incluye: Compiladores, Intérpretes, Editores de código y Entornos de Desarrollo Integrados (IDE) como Visual Studio.

El Rey del Software

El Sistema Operativo es el software más importante. Actúa como el administrador central de toda la computadora.

Drivers (Controladores)

Son programas pequeños que permiten que el Sistema Operativo se comunique con un hardware específico.

"Es el traductor que le explica al Windows cómo debe usar una impresora nueva o una tarjeta de video."

Firmware: Software Empotrado

Es un tipo de software que está grabado permanentemente en el hardware. Controla los circuitos electrónicos de un dispositivo.

Ejemplo: El BIOS de la placa madre.

Software Libre vs. Propietario

Libre / Open Source

Se puede ver el código, modificarlo y compartirlo gratis. (Ej: Linux, VLC).

Propietario

El código es secreto y generalmente requiere pago de licencia. (Ej: Windows, Photoshop).
