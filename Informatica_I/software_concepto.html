¿Qué es el Software?

Es el componente lógico de un sistema informático. Comprende el conjunto de los componentes lógicos necesarios que hacen posible la realización de tareas específicas.

  • Es la parte intangible (que no se puede tocar).
  • Envía instrucciones al Hardware para que funcione.
  • Incluye datos, programas y protocolos.

Software de Sistema

Es el software diseñado para servir de interfaz entre el usuario, las aplicaciones y el hardware. Su objetivo es desvincular al usuario de los detalles del sistema informático.

Sistemas Operativos

Software de Aplicación

Son los programas diseñados para facilitar la realización de tareas específicas en la computadora, como aplicaciones ofimáticas, bases de datos o videojuegos.

Ejemplos comunes:

  • Microsoft Word
  • Google Chrome
  • WhatsApp
  • Spotify

Software de Programación

Es el conjunto de herramientas que permiten al programador desarrollar programas de informática, utilizando diferentes alternativas y lenguajes de programación.

// Ejemplo de código simple
escribir "Hola Mundo";
leer nombre;
escribir "Bienvenido ", nombre;

Incluye: Editores de texto, Compiladores, Intérpretes y Entornos de Desarrollo (IDE).

El Sistema Operativo (S.O.)

Es el software principal o conjunto de programas de un sistema informático que gestiona los recursos de hardware y provee servicios a los programas de aplicación.

Funciones Vitales:

  • Gestión de la CPU (Procesos).
  • Gestión de la memoria RAM.
  • Gestión de entrada/salida (Periféricos).
  • Gestión de archivos.

Tipos de Interfaz de Usuario

GUI (Interfaz Gráfica)

Basada en ventanas, iconos, menús y puntero. Es amigable e intuitiva (Ej: Windows, macOS).

CLI (Línea de Comandos)

El usuario interactúa mediante texto y comandos específicos. (Ej: Terminal de Linux, CMD).

Microsoft Windows

Es el sistema operativo más utilizado en computadoras personales en el mundo. Desarrollado por Microsoft, se basa en una interfaz gráfica de usuario.

  • Versatilidad: Compatible con la mayoría del software.
  • Facilidad: Curva de aprendizaje muy baja.
  • Multitarea: Ejecuta varios programas a la vez.

GNU/Linux

Es un sistema operativo de código abierto (Open Source) y gratuito. Es muy robusto, seguro y preferido para servidores y por programadores.

  • Seguridad: Muy resistente a virus.
  • Personalización: Se puede modificar casi todo.
  • Variedad: Existen muchas "distribuciones" (Ubuntu, Fedora, Debian).

Sistemas Móviles y Especializados

Android / iOS

Diseñados para smartphones y tablets.

RTOS

Sistemas en tiempo real para maquinaria y robots.

BIOS / UEFI y Firmware

Es el primer software que se ejecuta al encender la computadora. Su función es preparar el hardware para que el Sistema Operativo pueda cargar.

POST (Power On Self Test):

Proceso de autocomprobación de encendido que verifica que la memoria, el teclado y el almacenamiento funcionen correctamente.

El Proceso de Arranque (Boot)

1 Suministro de energía a la placa base.
2 Ejecución del BIOS/UEFI (Prueba de Hardware).
3 Localización del Sistema Operativo en el Disco Duro.
4 Carga del Kernel en la memoria RAM.
