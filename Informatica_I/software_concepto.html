¿Qué es el Software?

Es el conjunto de programas, datos e instrucciones que permiten que el hardware realice tareas específicas.

Se conoce como la parte lógica o intangible de la computadora.

Importancia

Sin el software, el hardware es solo un conjunto de circuitos y piezas metálicas sin utilidad.

  • Permite la comunicación entre usuario y máquina.
  • Administra recursos físicos.
  • Ejecuta aplicaciones de productividad y ocio.

Clasificación

De Sistema

Controla el hardware.

De Programación

Para crear otros programas.

De Aplicación

Para tareas del usuario.

Software de Sistema

Es el software base que gestiona y controla el hardware.

Sistemas Operativos: Windows, macOS, Linux.
Controladores (Drivers): Para impresoras, tarjetas, etc.
Herramientas de Diagnóstico: Antivirus, optimizadores.

Software de Aplicación

Programas diseñados para que el usuario realice una tarea específica.

  • Microsoft Word
  • Google Chrome
  • WhatsApp
  • Videojuegos
  • Editores de fotos
  • Hojas de cálculo

El Sistema Operativo (SO)

Es el programa principal que se carga al encender la computadora.

Actúa como intermediario entre el hardware y el usuario final.

Funciones del SO

Gestión de Memoria

Administra el uso de la memoria RAM.

Gestión de Archivos

Crea, borra y organiza archivos en el disco.

Gestión de Procesos

Controla la ejecución de los programas.

Interfaz de Usuario

Permite al usuario interactuar visualmente.

Tipos de Interfaz

GUI (Gráfica)

Usa iconos, ventanas y puntero del mouse (Windows, Android).

CLI (Línea de Comandos)

Usa solo texto y comandos escritos (Terminal, CMD).

SO Móviles

Diseñados específicamente para pantallas táctiles y bajo consumo de energía.

Android

iOS

Seguridad

El software también incluye herramientas para proteger el sistema:

  • Antivirus (protección contra malware).
  • Firewall (control de tráfico de red).
  • Actualizaciones (parches de seguridad).
