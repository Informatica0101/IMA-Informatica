¿Qué es el Software?

Es el conjunto de programas, instrucciones y reglas informáticas para ejecutar tareas en una computadora. Es la parte lógica e intangible.

Clasificación del Software

De Sistema

Permite que el hardware funcione (SO).

De Aplicación

Tareas específicas para el usuario (Word).

De Programación

Herramientas para crear software (Python).

Software de Sistema

Su objetivo es desvincular al usuario y al programador de los detalles de la computadora en particular que se use.

  • Sistemas Operativos (Windows, Linux)
  • Controladores de dispositivos (Drivers)
  • Herramientas de diagnóstico

Software de Aplicación

Es aquel que permite a los usuarios llevar a cabo una o varias tareas específicas.

Ofimática (Office)
Navegadores (Chrome)
Videojuegos
Editores de imagen

Software de Programación

Conjunto de herramientas que permiten al programador desarrollar programas de informática, usando diferentes alternativas y lenguajes.

Python, C++, Java, JavaScript

Licencias de Software

  • Propietario: De pago, código cerrado (Windows).
  • Libre: Se puede usar, copiar y modificar (Linux).

Software en la Nube

Programas que se ejecutan en Internet sin necesidad de instalación local. Ejemplo: Google Docs, Office 365.

Instalación y Desinstalación

El proceso de configurar el software en el disco duro para que el SO lo reconozca y gestione sus recursos.

Versiones y Actualizaciones

Mejoras continuas del software para corregir errores (bugs) o añadir nuevas funcionalidades.

El Código Fuente

Es el texto escrito en un lenguaje de programación que define qué hace el programa. Los humanos lo leen, las máquinas lo ejecutan.
