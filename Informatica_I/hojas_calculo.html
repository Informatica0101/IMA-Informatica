¿Qué es una Hoja de Cálculo?

Es una aplicación diseñada para organizar, manipular y analizar datos numéricos mediante una cuadrícula de filas y columnas.

Principal exponente: Microsoft Excel.

La Celda

Es la unidad básica de la hoja de cálculo. Se forma en la intersección de una fila y una columna.

Su nombre viene de la Columna (Letra) y la Fila (Número). Ej: A1.

Filas y Columnas

Columnas

Identificadas con Letras (A, B, C...). Son verticales.

Filas

Identificadas con Números (1, 2, 3...). Son horizontales.

Tipos de Datos

En una celda podemos ingresar:

  • Texto: Nombres, etiquetas, descripciones.
  • Números: Valores para cálculos.
  • Fórmulas: Expresiones que realizan cálculos.

Las Fórmulas

Toda fórmula en Excel DEBE comenzar con el signo igual (=).

Ejemplo: =A1 + B1

Operadores Básicos

+ (Suma)
- (Resta)
* (Mult)
/ (Div)

Las Funciones

Son fórmulas predefinidas que simplifican tareas.

  • =SUMA(rango): Suma todos los valores.
  • =PROMEDIO(rango): Calcula la media aritmética.
  • =MAX(rango): Valor más alto.

Rangos

Un rango es un conjunto de dos o más celdas. Se identifica por la celda inicial y final separadas por dos puntos (:).

Ejemplo: A1:A10

Gráficos

Permiten representar los datos de forma visual para facilitar su análisis e interpretación.

Importancia en el Trabajo

Lógica de Cálculo

FUNCTION CalcularTotal(celdas[])

  LET suma = 0

  FOR EACH valor IN celdas

    suma = suma + valor

  ENDFOR

  RETURN suma

ENDFUNCTION
