¿Qué es una Hoja de Cálculo?

Es una aplicación diseñada para organizar, analizar y manipular datos numéricos y alfanuméricos en forma de tablas.

Permite realizar cálculos complejos mediante el uso de fórmulas y funciones de manera automatizada.

Celdas, Filas y Columnas

La estructura fundamental de una hoja de cálculo es la cuadrícula.

  • Columnas: Se identifican por letras (A, B, C...).
  • Filas: Se identifican por números (1, 2, 3...).
  • Celda: La intersección de una fila y una columna (Ej: B4).

Tipos de Datos en Celdas

Valores Constantes

Datos que no cambian (Texto, números, fechas).

Fórmulas

Secuencias que calculan nuevos valores a partir de otros.

Operadores Aritméticos

Para realizar cálculos en Excel u otras hojas de cálculo, se utilizan los siguientes símbolos:

+ (Suma)
- (Resta)
* (Multiplicación)
/ (División)

Nota: Todas las fórmulas deben comenzar con el signo igual (=).

Funciones Predeterminadas

Son fórmulas ya creadas por la aplicación para simplificar tareas comunes.

  • =SUMA(rango): Suma todos los valores.
  • =PROMEDIO(rango): Calcula la media aritmética.
  • =MAX(rango): Devuelve el valor más alto.
  • =MIN(rango): Devuelve el valor más bajo.

Formato de Datos

Podemos cambiar cómo se ven los datos según su naturaleza:

Moneda
Porcentaje
Fecha
Contabilidad

Referencias Relativas y Absolutas

Relativas

Cambian al copiar la fórmula a otra posición (Ej: A1).

Absolutas

Se mantienen fijas usando el signo $ (Ej: $A$1).

Visualización de Datos (Gráficos)

Permiten representar los datos numéricos de forma visual para facilitar su interpretación.

Organización de la Información

Filtros: Permiten mostrar solo las filas que cumplen con ciertos criterios.

Ordenamiento: Organiza los datos de forma alfabética (A-Z) o numérica de mayor a menor.

Manejo de Errores

Cuando algo sale mal en una fórmula, la aplicación nos avisa con códigos:

#¡DIV/0!
División por cero.
#¿NOMBRE?
Función mal escrita.

Ejercicio Guiado: Factura Simple

Si la celda B1 tiene el precio (100) y la celda B2 tiene la cantidad (5):

¿Cuál es la fórmula para el total?

Respuesta: =B1*B2
