¿Qué es una Hoja de Cálculo?

Es un tipo de documento que permite manipular datos numéricos y alfanuméricos dispuestos en forma de tablas compuestas por celdas, las cuales se suelen organizar en una matriz de filas y columnas.

Es la herramienta por excelencia para cálculos matemáticos, estadísticos y financieros.

Microsoft Excel

Es el software de hoja de cálculo líder en la industria, desarrollado por Microsoft. Su potencia radica en la capacidad de automatizar cálculos mediante fórmulas y funciones.

  • Formato nativo: .xlsx
  • Soporta millones de registros por hoja.

Columnas, Filas y Celdas

Columnas

Se identifican con Letras (A, B, C...). Son verticales.

Filas

Se identifican con Números (1, 2, 3...). Son horizontales.

Celdas

Intersección de una fila y una columna. (Ej: B5).

¿Qué podemos escribir?

Excel reconoce automáticamente el tipo de contenido que ingresamos en una celda para saber cómo procesarlo.

  • Texto: Nombres, descripciones.
  • Números: Valores para cálculos.
  • Fechas: Días, meses y años.
  • Fórmulas: Instrucciones matemáticas.

El Poder de las Fórmulas

Una fórmula es un código que introducimos en una celda para realizar un cálculo y devolver un resultado.

REGLA DE ORO

Todas las fórmulas en Excel deben comenzar obligatoriamente con el signo igual: =

Ejemplo: =A1+B1

Operadores Básicos

Excel utiliza los símbolos estándar para las operaciones aritméticas:

Suma (+ )
Resta (- )
Mult (* )
Div (/ )

Nota: Para potencia se usa el acento circunflejo ( ^ ).

Funciones Predeterminadas

Las funciones son fórmulas simplificadas que ya vienen creadas en Excel para realizar tareas complejas de forma rápida.

Sintaxis: =NOMBRE_FUNCION(rango)

  • =SUMA(A1:A50): Suma todos los valores en ese rango.
  • =PROMEDIO(B1:B10): Calcula la media aritmética.

Los Rangos

Un rango es un conjunto de celdas contiguas seleccionadas. Se expresa mencionando la primera celda y la última separadas por dos puntos ( : ).

A1 : C10

Abarca desde A1 hasta C10

Estética de la Hoja

Para que los datos sean legibles, podemos aplicar formatos visuales:

  • Formato Moneda: Agrega el signo $ o L y separador de miles.
  • Formato Porcentaje: Multiplica por 100 y añade el signo %.
  • Relleno de Celda: Colores de fondo para títulos.
