Introducción al Formato de Página

El formato de página define la apariencia general y la estructura física de un documento. Es vital para la legibilidad, la presentación profesional y la preparación para la impresión.

Impacto Visual

Hace que el documento sea fácil de leer y atractivo.

Componentes Esenciales del Formato de Página

Estas configuraciones se encuentran usualmente en la pestaña "Disposición" o "Formato".

  • Márgenes: Espacio en blanco alrededor de los bordes del documento (Superior, Inferior, Izquierdo, Derecho).
  • Orientación: Dirección en la que se imprime la página (Vertical o Horizontal).
  • Tamaño: Define las dimensiones del papel (ej. Carta - 8.5 pulgadas x 11 pulgadas, Oficio - 8.5 pulgadas x 14 pulgadas).
  • Saltos de Página: Controlan dónde termina una página y dónde comienza la siguiente.
  • Columnas: Dividir el texto en dos o más secciones verticales (típico en periódicos).

Ejemplos Visuales: Márgenes y Orientación

La combinación correcta mejora la estética y el uso del espacio.

Margen Estrecho (Más contenido)

Este margen maximiza el espacio para el texto. Útil para informes extensos.

1. La Herramienta de Inserción de Imágenes

Insertar imágenes en un documento de texto no es solo "pegar" una foto. Requiere entender cómo el programa de Ofimática gestionará el espacio y la interacción de la imagen con el texto.

Definición

Proceso de incorporar archivos visuales (como .jpg, .png, .gif) a un documento para complementar o ilustrar el contenido textual.

2. El Secreto: Ajuste de Texto (Text Wrapping)

Esta propiedad define cómo el texto que rodea a la imagen se va a posicionar. Es la diferencia entre un documento fluido y uno desordenado.

Opción Clave: Cuadrado/Estrecho

El texto rodea la imagen de manera ajustada. Permite que la imagen se coloque en medio de un párrafo.

3. Posicionamiento y Anclaje

Una vez que el Ajuste de Texto no es "En línea", podemos posicionar la imagen con respecto a la página o al párrafo.

Anclaje

Es el símbolo de un ancla que aparece en la imagen. Indica el párrafo al que la imagen está ligada. Si ese párrafo se mueve a otra página, la imagen lo seguirá.

Uso Avanzado: Creación de un Índice

Un índice o tabla de contenido automático es fundamental para documentos largos (Tesis, Informes).

El Secreto: Estilos de Título

  1. Aplicar Estilos: Marcar los encabezados del documento con los estilos Título 1, Título 2, etc. (Ej. Título 1 para Capítulos, Título 2 para Secciones).
  2. Insertar Índice: Ir a la pestaña Referencias (o similar) y seleccionar "Tabla de Contenido".
  3. Actualizar: Hacer clic derecho sobre el índice y seleccionar "Actualizar Campos" cada vez que se modifiquen los títulos o números de página.

Guía Práctica: Configurar una Página

Pasos para Configuración de Página (Ej. Normas APA):

  1. Márgenes: Ir a Formato/Disposición y seleccionar "Márgenes Personalizados". Establecer 1 pulgada (2.54 cm) en los cuatro lados (Superior, Inferior, Izquierdo, Derecho).
  2. Tamaño: Asegurar que el tamaño de papel sea Carta (8.5 pulgadas).
  3. Fuente: En la pestaña Inicio, establecer el tipo de fuente (ej. Times New Roman) y tamaño (12 puntos).
